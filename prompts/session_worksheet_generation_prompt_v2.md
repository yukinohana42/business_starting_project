# Session Worksheet Generation Prompt v2

Use this prompt to create worksheets for any session.

## Required Reading

- `SOURCE_OF_TRUTH.md`
- `DESIGN.md`
- `docs/visual_learning_system_v2.md`
- `docs/global_visual_acceptance_criteria.md`
- `docs/session_visual_design_map_v2.md`
- Existing worksheet for the target session

## Worksheet Definition

A worksheet is not a test. It is a thinking surface.

It should help the child write observations, arrange evidence, describe Before/After, plan a small test, and reflect in their own words.

## Required Sections

Include only sections that fit the target session:

- Mission
- Observation Log
- Before / After
- Who changes?
- What changes?
- Card Table memo
- Question list
- Small experiment
- Reaction log
- Reflection stamp
- Final Mission

## Design Rules

- Large writing spaces.
- Short labels.
- No model-answer feel.
- Parent prompts are small and supportive.
- Printable A4 friendly.
- Zoom use can be supported by numbering fields.

## Prohibitions

- Correct-answer blanks.
- Parent scoring.
- Business plan heaviness.
- Too many fields.
- Reward or badge collection as the goal.

## Acceptance Check

Verify:

- The child can write in their own words.
- The worksheet asks who changes and how.
- It supports observation, question, test, or reflection.
- It does not make the parent a grader.

# スライド生成プロンプト

あなたは中学2年生向けの起業教育スライドを書く教材制作者です。

次の条件でMarkdownスライドを作ってください。

- 1スライド1メッセージ
- スライド区切りは `---`
- 子供向けの言葉にする
- 専門用語は必ず言い換える
- 身近な例を入れる
- 問いかけを多くする
- 文字を詰め込みすぎない
- 起業を押し付けない

参照するファイル:

- `SOURCE_OF_TRUTH.md`
- `DESIGN.md`
- `docs/visual_learning_system_v2.md`
- `docs/global_visual_acceptance_criteria.md`
- `docs/session_visual_design_map_v2.md`
- 対象セッションの `README.md`
- 対象セッションの `slide_outline.md`

出力はMarkdownだけにしてください。

## v2追加条件

子供向けスライドの場合、最初に各スライドを次のどれかに分類してください。

- Mission Board
- Quest Map
- Detective Note
- Case File
- Evidence Board
- Card Table
- Experiment Lab
- Pitch Stage
- Final Mission Sheet
- Reflection Log

各スライドごとに、キャラクター行動、ゲーム要素、子供がする行動、相手の変化を定義してから本文を書いてください。

丸アイコンだけ、ゲーム用語だけ、カードを並べただけのスライドは禁止です。不合格なら生成を止め、修正案を出してください。

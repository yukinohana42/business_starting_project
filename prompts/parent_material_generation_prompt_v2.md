# Parent Material Generation Prompt v2

Use this prompt for parent-facing slides, guides, checklists, question cards, or scripts.

## Required Reading

- `SOURCE_OF_TRUTH.md`
- `docs/parent_material_design_principles_v2.md`
- `docs/global_visual_acceptance_criteria.md`
- Existing parent material for the target session

## Core Rule

Parent materials should not be child-facing game screens. They should be warm, calm, practical facilitator tools.

## Required Classification

Classify the material as:

- Facilitator Guide
- Parent Briefing
- Question Card
- OK/NG Response Sheet
- Observation Log
- Safety Boundary Card
- Reflection Support Sheet
- Praise Guide
- Session Flow Map

## Required Content

Include:

- Parent role: facilitator, not teacher, scorer, or game master.
- What the parent says.
- What the parent avoids.
- Good questions.
- OK/NG voice examples.
- Safety boundaries.
- Praise guide focused on observation, questions, trials, and improvement.

## Prohibitions

- Anxiety-based motivation.
- Entrepreneurship pressure.
- Money hype.
- Comparing siblings or children.
- Scoring the child.
- Turning badges into rewards.

## Acceptance Check

Before finalizing, verify:

- The parent can use it during the session.
- It gives concrete words, not only philosophy.
- It protects the child from scoring and pressure.
- It stays warm without becoming a child-game UI.

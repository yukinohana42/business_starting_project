# カード生成プロンプト

あなたは家庭内ワークショップで使う印刷用カード教材の制作者です。

次の条件でカード案を作ってください。

- 中学2年生が見てすぐ意味がわかる
- 1カード1テーマ
- ランダムに引いてワークに使える
- 身近な例を多くする
- 問いカードには考えが深まる質問を入れる
- A4印刷で切りやすい形に後で整えられる構造にする

カードの種類:

- 価値カード
- 問題カード
- 顧客カード
- アイデアカード
- PMF信号カード
- ピッチ問いカード

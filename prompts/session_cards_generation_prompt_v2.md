# Session Cards Generation Prompt v2

Use this prompt to create cards for any session.

## Required Reading

- `SOURCE_OF_TRUTH.md`
- `docs/visual_learning_system_v2.md`
- `docs/global_visual_acceptance_criteria.md`
- `docs/session_visual_design_map_v2.md`
- Existing cards for the target session

## Card Definition

A card is not decoration. A card is a tool that makes a child choose, arrange, compare, write, or talk.

## Required Fields Per Card

For each card, include:

- Card name
- Learner action
- Front text
- Back or parent prompt
- What the child writes, chooses, or says
- Which session mission it supports
- Safety note

## Allowed Card Types

- Mission Card
- Problem Card
- Observation Card
- Question Card
- Customer Card
- Idea Card
- Value Change Card
- Experiment Card
- PMF Signal Card
- Reaction Memo Card
- Pitch Card
- Reflection Card

## Prohibitions

- Winner cards.
- Ranking cards.
- Earn-more cards.
- Parent scoring cards.
- Cards that only display a concept and do not create action.
- Existing IP imitation.

## Acceptance Check

Before finalizing, verify:

- 1 card = 1 action.
- The child can use the card without a lecture.
- The card does not imply a correct answer.
- The parent uses it to ask, not to score.
- It links to observation, question, trial, or reflection.

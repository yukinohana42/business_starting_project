# Visual Acceptance Review Prompt v2

Use this prompt to review any generated visual material before proceeding to HTML/CSS, PNG, PDF, PPTX, 4:5, or print layout.

## Required Reading

- `docs/visual_learning_system_v2.md`
- `docs/global_visual_acceptance_criteria.md`
- `docs/session_visual_design_map_v2.md`
- `docs/parent_material_design_principles_v2.md`

## Review Targets

Review:

- Infographics
- Slides
- Cards
- Worksheets
- Parent materials
- Badges
- Stamps
- Quest maps
- Contact sheets

## Required Table

Create a table:

| Page/Item | Character action | Game element function | Learner/parent action | Visible change | Safety | Readability | Verdict |
|---|---|---|---|---|---|---|---|

Verdict must be Pass, Needs revision, or Stop.

## Stop Conditions

Mark Stop if:

- Character is a round icon, face only, name label, or standing only.
- Game element is title text only.
- The screen is heading + body + card list only.
- The child only reads and does not see what to do.
- Parent is framed as teacher, scorer, or game master.
- Money, ranking, scoring, reward, trophy, or winner framing appears.
- Existing IP imitation risk appears.

## Required Output

Output:

1. Findings ordered by severity.
2. Pass/Needs revision/Stop table.
3. Minimum revisions required.
4. Whether production may proceed.

Do not approve production if the material is merely readable but not a discovery-game learning experience.

# Codex Prompt: Design Brushup Plan

Use this prompt to plan the next design pass for the junior entrepreneurship workshop.

## Goal

Turn the existing educational content into a more engaging, visual, game-like experience without changing the core philosophy.

## Direction

- Follow `SOURCE_OF_TRUTH.md` and `docs/gpt-fixed-context-2026-06-01/03_DESIGN_PRINCIPLES.md`.
- Treat the workshop as a discovery game, not a quiz with one correct answer.
- Use cards, missions, points, levels, badges, and small choices where they help students move.
- Use middle-school-aged characters, familiar scenes, and short stories.
- Show value as a visible change in another person:困った to 助かった, 面倒 to ラク, 退屈 to 楽しい.
- Do not imitate existing manga, anime, game characters, logos, or proprietary art styles.

## Expected Output

Create a concrete design improvement plan for slides, cards, worksheets, and facilitation scripts. Keep the plan practical enough for the next Codex task to implement.

# visual-storytelling-designer

## 用途

スライド、カード、ワークシート、インフォグラフィックで、言葉だけでは伝わりにくい概念を図解や人物の変化で見せるときに使う。

## 必ず読むもの

- `SOURCE_OF_TRUTH.md`
- `DESIGN.md`
- `docs/visual_storytelling_guidelines.md`
- `docs/manga_panel_usage_guidelines.md`
- `docs/character_and_illustration_guidelines.md`
- 対象教材の原稿

## 出力形式

- 伝える概念
- 抽象語の中学生向け言い換え
- Before/Afterの見せ方
- 図解要素
- 人物イラストの役割
- 吹き出し案
- 1ページの文字量目安
- Zoomでの読みやすさ
- 4:5展開時の注意

## ルール

- 価値は、言葉だけでなく人物の変化で見せる。
- 困った -> 助かった、面倒 -> ラク、退屈 -> 楽しいを基本にする。
- 1ページ1メッセージを守る。
- 人物は基本1から2人、多くても3人までにする。
- 1ページに入れるイラストは、メイン1つ、補助1から2つまでにする。
- マンガ的な1コマ表現は理解補助として使う。
- 既存漫画作品の表現やキャラクターを模倣しない。

## 品質基準

- 中学2年生に伝わる。
- 親が横で見ても安心できる。
- 起業を押し付けていない。
- 不安を煽っていない。
- 文字、図解、人物の役割が整理されている。

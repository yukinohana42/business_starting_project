# facilitator-script-writer

## 用途

親が当日読める進行台本を作る。

## 必ず読むもの

- `SOURCE_OF_TRUTH.md`
- `DESIGN.md`
- `docs/facilitation_guide.md`
- 対象セッションの `README.md`

## 台本のルール

- 親が話しすぎない
- 5から10分ごとに子供が話す、書く、選ぶ時間を入れる
- 否定せずに問いで深める
- 「それは本当に使う？」「誰が困っている？」「今はどうしている？」を多用する
- 最後にまとめと次回へのつなぎを入れる

## 出力形式

- 時間配分
- 親の読み上げ文
- 子供への問い
- 想定回答
- 深掘り質問
- ワーク指示
- まとめ

# character-illustration-designer

## 用途

教材に人物イラストやオリジナルキャラクターを入れるときに使う。

人物イラストは飾りではなく、価値や変化を伝えるために使う。抽象的な概念を、表情、姿勢、行動の変化として見せる。

## 必ず読むもの

- `SOURCE_OF_TRUTH.md`
- `DESIGN.md`
- `docs/character_and_illustration_guidelines.md`
- `docs/visual_storytelling_guidelines.md`
- `docs/manga_panel_usage_guidelines.md`
- 対象教材の原稿

## 出力形式

- 登場キャラクター
- キャラクターの役割
- 人物イラストの内容
- Before/Afterの変化
- 表情、姿勢、行動
- 小物
- 吹き出し案
- 親が見ても安心できる表現
- 避けるべき表現

## ルール

- 完全オリジナルキャラクターにする。
- 既存作品、既存キャラクター、特定漫画家の絵柄を真似しない。
- 中学2年生が感情移入しやすく、幼稚にしすぎない。
- 親が見ても安心できる表現にする。
- 会社員や起業しない選択を悪く見せない。
- お金を派手に煽らない。
- Beforeは困りごとを見せるが、相手を笑いものにしない。
- Afterは少し良くなった状態として見せる。

## 品質基準

- 価値が「人の表情・行動・状態の変化」として見える。
- イラストが説明を邪魔していない。
- 吹き出しが短い。
- Zoomで見ても意味が伝わる。
- 既存IP模倣の指示がない。

# workshop-designer

## 用途

新しいワークショップ回を設計する、または既存回を30分版、45分版、60分版に調整するときに使う。

## 必ず読むもの

- `SOURCE_OF_TRUTH.md`
- `DESIGN.md`
- `docs/facilitation_guide.md`
- 対象セッションの `README.md`

## 出力形式

- 目的
- 対象
- 伝える概念
- 導入の問い
- メインワーク
- 親の問いかけ
- 子供の作業
- まとめ
- 宿題
- 必要なカードやワークシート

## 品質基準

- 中学2年生に伝わる言葉で書く
- 親の講義時間を短くする
- 子供が考える、話す、書く、選ぶ時間を作る
- 起業を押し付けない
- 小さく試す行動につなげる

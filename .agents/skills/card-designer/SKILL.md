# card-designer

## 用途

印刷用カード教材を作る。

## 必ず読むもの

- `SOURCE_OF_TRUTH.md`
- `DESIGN.md`
- `cards/printable_cards_index.md`

## カードの種類

- 価値カード
- 問題カード
- 顧客カード
- アイデアカード
- PMF信号カード
- ピッチ問いカード

## ルール

- 1カード1テーマ
- 中学生がすぐ意味をわかる言葉にする
- ランダムに引いて使える形にする
- 問いカードには「誰が」「いつ」「今はどうしている」を入れる
- A4印刷で切りやすい形に後で整えられるようにする

# 第4回 進行台本

## 0. 前回の基本文確認 5分

前回作った基本文を声に出して読む。

> これは、______ な人が、______ で困っているときに、______ できるサービスです。

## 1. PMFを説明する 10分

PMFは「作ったものと欲しい人の気持ちがピタッとはまること」。

良いアイデアと、実際に使われるアイデアは違う。

問いかけ:

- 無料なら使う人は、本当に欲しい人？
- 面白いけど使わないものはある？
- 本当に欲しいものは、どんな反応になる？

## 2. PMF信号で見る 15分

青信号:

- 相手が自分から使いたがる
- 友達に勧める
- 2回目も使う
- お金や時間を使ってもよいと言う

黄信号:

- 面白いけど続かなそう
- 説明すると反応する
- 便利そうだが今は困っていない

赤信号:

- 反応が薄い
- 今の方法で満足している
- 誰が困るかはっきりしない

赤信号は失敗ではなく、「この方向ではない」とわかった学び。

## 3. 小さな実験を設計する 20分

実験は、完璧な商品を作る前に行う。

例:

- 紙の説明だけ見せる
- 友達に1回だけ使ってもらう
- LINEでチェックリストを送る
- 3日間だけ手伝う
- サンプル動画を1本作る

## 4. 次に直すところ 10分

実験で何を見るかを決める。

- 使ったか
- 2回目も使いたいか
- どこで止まったか
- 何が面倒だったか

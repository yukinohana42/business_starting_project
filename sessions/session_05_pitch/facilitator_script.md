# 第5回 進行台本

## 0. これまでの復習 10分

問いかけ:

- 仕事とは何だった？
- 価値はどこにあった？
- 誰の何を解決する案になった？
- 小さく試すなら何をする？

## 1. ピッチの型を説明する 10分

3分ピッチは、すごいことを言う場ではない。

誰のどんな問題を、どう解くかが伝わることが大事。

型:

1. 僕たちが見つけた問題は、______ です。
2. 困っている人は、______ です。
3. 今は ______ で解決していますが、______ という不満があります。
4. そこで僕たちは、______ を考えました。
5. これを使うと、______ になります。
6. まずは ______ で試してみます。

## 2. 原稿を作る 20分

ワークシートを使って3分原稿を書く。

親は「もっとすごく」ではなく「誰に何が伝わるか」を見る。

## 3. 発表 15分

子供が発表する。親は審査員として質問する。

質問例:

- 誰が一番困っている？
- 今の方法では何がダメ？
- 本当に使いたい人はどうわかる？
- 小さく試すなら何から？

## 4. セミナーで見るポイント 10分

起業セミナーでは、次を見る。

- 他の人はどんな問題を見つけているか
- 顧客をどう具体的にしているか
- どうやって試そうとしているか
- どんな質問を受けているか

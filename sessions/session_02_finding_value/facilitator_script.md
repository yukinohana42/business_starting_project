# 第2回 進行台本

## 0. 前回の復習 5分

問いかけ:

- 仕事とは何を何に変えることだった？
- お金は何への対価だった？

## 1. 価値の種を探す 10分

親が短く説明する。

ビジネスの種は、すごい発明よりも、日常の小さな違和感にある。

探す場所:

- イラッとしたこと
- 面倒だったこと
- 好きだけど、もっと良くしたいこと
- もっと楽しくしたいこと
- 何度も困ること

## 2. 最近の困りごとを出す 15分

子供に10個書いてもらう。小さくてよい。

親は「それは本当に困っているの？」と決めつけず、次を聞く。

- いつ起きる？
- 誰が困る？
- 今はどうしている？
- それがなくなったらうれしい？

## 3. インタビュー質問を作る 20分

聞く相手を3人決める。

質問は、未来の希望ではなく、過去の行動を聞く。

良い質問:

- 最近それで困ったことはある？
- 今はどうやって解決している？
- それがどうなったらうれしい？

避けたい質問:

- こういうアプリがあったら使う？
- いくらなら払う？

## 4. 宿題確認 5分

次回までに、家族、友達、先生など安全な相手に3人インタビューする。

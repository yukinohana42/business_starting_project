# 第3回 進行台本

## 0. インタビュー共有 10分

子供に、聞いてきた内容を話してもらう。

問いかけ:

- 意外だった答えは？
- 自分の予想と違ったところは？
- 一番困っていそうだった人は？

## 1. 顧客を具体的にする 10分

「みんな」ではなく、具体的な1人を思い浮かべる。

例:

- テスト前に英単語が覚えられない中学生
- 部活の後に疲れて勉強できない中学生
- 朝起きるのが苦手な人

## 2. 解決案をたくさん出す 15分

10分で20個を目標に出す。変な案もOK。

親は評価を急がず、量を出させる。

使える型:

- アプリにする
- チェックリストにする
- 友達と競争にする
- ポイント制にする
- 親に通知する
- 動画にする
- 代わりにやる
- 3日間チャレンジにする

## 3. 選ぶ 15分

3つの基準で点数をつける。

- 困り度: 本当に困っているか
- 頻度: 何度も起きるか
- 試しやすさ: 明日小さく試せるか

合計点が高い案を1つ選ぶ。

## 4. 基本文を作る 10分

次の文に入れる。

> これは、______ な人が、______ で困っているときに、______ できるサービスです。

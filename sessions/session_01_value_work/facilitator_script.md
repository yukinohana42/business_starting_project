# 第1回 進行台本

## 0. はじめに 5分

今日は「起業家になる方法」ではなく、「仕事って何なのか」を考える回にする。

問いかけ:

- 一番ラクにお金をもらっていそうな仕事って何？
- YouTuber、ゲーム実況者、プロスポーツ選手、ラーメン屋、コンビニ、塾の先生は、何でお金をもらっていると思う？

## 1. 仕事の正体 10分

親が短く説明する。

仕事は「物を売ること」だけではない。誰かの状態を良くすること。

例:

- ラーメン屋: お腹が空いたを満足に変える
- コンビニ: 欲しいものが今すぐないを、すぐ買えるに変える
- ゲーム実況者: 退屈を楽しいに変える
- 塾: わからないを、わかるかもに変える

深掘り質問:

- その仕事は、誰をどう助けている？
- その人は何にお金を払っている？

## 2. 価値変換ゲーム 20分

カードや紙に、困りごとを書いて並べる。

子供に「変換後」を考えさせる。

例:

- 困った → 助かった
- 面倒 → ラク
- 退屈 → 楽しい
- 不安 → 安心
- わからない → できそう

親は正解を言わず、「それは誰がうれしい？」と聞く。

## 3. まとめ 10分

子供に1文で書いてもらう。

> 仕事とは、______ を ______ に変えること。

最後に聞く:

- 今日見方が変わった仕事はある？
- 起業セミナーで、どんなところを見てみたい？

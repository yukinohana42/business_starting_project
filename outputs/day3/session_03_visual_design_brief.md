# Day3 Visual Design Brief

## Session

Day3: Idea Builder / アイデアビルダー

## Mission

誰の何を助けるかを形にせよ。

## Learning Goal

良いアイデアは、相手の変化が見える形で考えるものだとわかる。

## Main Page Types

- Mission Board
- Idea Workshop
- Card Table
- Customer-Problem-Solution Board
- Sketch Sheet
- Final Mission Sheet

## Game Elements

- Customer Card
- Problem Card
- Solution Card
- Value Change Card
- Idea Builder Board
- Draft Stamp

## Character Roles

- Sora arranges cards and sketches solution ideas.
- Yuu checks whether the idea shows who changes and how.
- Miho offers a Question Card when the idea becomes too vague.

## Learner Actions

- 選ぶ: Customer Cardを1つ選ぶ。
- 並べる: Problem CardとSolution Cardをつなげる。
- 比べる: 相手の変化が見える案を選ぶ。
- 書く: 1文で「誰の何を何に変えるか」を書く。
- 話す: なぜその人が助かるかを言う。

## Visible Changes

- ひとりで困る -> 助けてもらえる
- わからない -> できそう
- 面倒 -> ラク
- つまらない -> 楽しい

## Image Generation Notes

Generate an active workshop table, not a static explanation chart. Show Sora's hands moving cards, a rough sketch sheet, sticky notes, and Yuu pointing to the "who changes?" area.

Keep exact Japanese text, card labels, and blanks for later layout. The generated image should leave clean card surfaces.

Avoid making the scene look like a startup pitch contest or business-plan classroom.

## Suggested Page Sequence

| Page | Type | Visual Core | Learner Action |
| --- | --- | --- | --- |
| 1 | Mission Board | Sora receives Idea Builder mission | Confirm mission |
| 2 | Customer Card Board | Several concrete people cards | Choose one person |
| 3 | Card Table | Problem and Solution cards are arranged | Make combinations |
| 4 | Sketch Sheet | Sora sketches a small solution | Draw or describe |
| 5 | Value Change Board | Idea is checked by Before/After | Compare changes |
| 6 | Final Mission Sheet | "私たちは、＿＿の＿＿を＿＿に変える。" | Write one sentence |

## Negative Prompt

- "best idea" ranking
- money evaluation
- investor pitch mood
- huge business plan
- trophy or score
- face-only Sora
- generic table slide

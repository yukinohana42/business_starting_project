# 第1回インフォグラフィック v2 ストーリーボード

## 全体方針

既存の6枚構成を活かし、各ページに「ミッション」「人物の変化」「短い問い」を追加する。キャラクターは完全オリジナルとし、既存作品の絵柄やキャラクターを模倣しない。

## 01 / 06 今日のミッション

- 役割: 導入。
- ミッション: お金と仕事の正体を見破れ。
- 主役: 価値探偵ユウ。
- 画面: ユウが探偵ノートを開き、仕事・お金・価値の3カードを見る。
- 図解: 仕事 -> お金 -> 価値の3カード。今日のゴール文。
- 吹き出し: 「相手がどう変わるかを見てみよう」
- 次へ: まず仕事を変化で見る。

## 02 / 06 仕事って何？

- 役割: 仕事の定義。
- ミッション: 困った顔を助かった顔に変えろ。
- 主役: ユウ、困っている同世代キャラ。
- 画面: Beforeは困っている表情、Afterは少し安心した表情。
- 図解: 困った -> 助かった、面倒 -> ラク、退屈 -> 楽しい。
- 吹き出し: 「誰が助かった？」
- 次へ: 変化が届くと、お金はどう見えるか。

## 03 / 06 お金って何？

- 役割: お金の扱い方。
- ミッション: ありがとうの正体を見つけろ。
- 主役: ユウ、助かった人。
- 画面: 価値が届き、ありがとうカードが生まれ、お金アイコンにつながる。
- 図解: 価値が届く -> ありがとう -> お金。
- 吹き出し: 「お金は、ありがとうが見える形」
- 次へ: 価値とは何かを見る。

## 04 / 06 価値って何？

- 役割: 価値の定義。
- ミッション: 変化カードを完成させろ。
- 主役: ユウ、アイデアビルダー・ソラ。
- 画面: 2人が3枚の価値変換カードを並べる。
- 図解: 3つのBefore/Afterカード。
- 吹き出し: 「すごそうより、変わったか」
- 次へ: 身近な仕事を探偵する。

## 05 / 06 ビジネス探偵ミッション

- 役割: 身近な例。
- ミッション: この仕事は、誰をどう変えている？
- 主役: ユウ、ソラ。
- 画面: 探偵ノートに、コンビニ、ラーメン屋、塾、動画、スマホゲームの変化をメモする。
- 図解: ビジネス探偵カード。
- 吹き出し: 「何を売るかより、どう変わるか」
- 次へ: 見つけた変化を1文にする。

## 06 / 06 Final Mission

- 役割: まとめ。
- ミッション: 仕事の正体を1文で言え。
- 主役: ユウ、ソラ、ナビゲーター・ミホ。
- 画面: ワークシートにFinal Missionを書き込む。
- 図解: 困った/面倒/退屈 -> 助かった/ラク/楽しい。
- 吹き出し: 「自分の言葉でOK」
- 次へ: Day2では日常の困りごとから価値の種を探す。

## レイアウト注意

- 16:9では人物を右側またはカード横に置く。
- 4:5では人物とカードを縦積みにする。
- 吹き出しは1ページ1つまで。
- 人物は1ページ1から2人、Page 6のみ3人まで。
- 細かい表情より、姿勢とBefore/Afterが見える構成を優先する。

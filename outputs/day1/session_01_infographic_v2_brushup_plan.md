# 第1回インフォグラフィック v2 ブラッシュアップ計画

## 目的

既存の6枚構成を保ちながら、ゲーミフィケーション、同世代オリジナルキャラクター、Before/After表現、マンガ的な1コマ表現を加える。PNG再生成はこの段階では行わない。

## v2で変えること

- Page 1を「今日のミッション」化する。
- Page 5を「ビジネス探偵ミッション」化する。
- Page 6を「Final Mission」化する。
- 価値探偵ユウ、アイデアビルダー・ソラ、ナビゲーター・ミホを完全オリジナルキャラクターとして使う。
- 価値を「人の表情・姿勢・行動の変化」で見せる。
- 吹き出しを短く入れ、理解補助として使う。

## 変えないこと

- 起業を押し付けない。
- お金を煽らない。
- 勝敗やランキングにしない。
- 既存作品や特定漫画家の絵柄を模倣しない。
- PNGや画像はこの段階で再生成しない。
- 既存6枚の説明順を大きく崩さない。

## ページ別ブラッシュアップ方針

### Page 1

- 現状: 仕事・お金・価値の導入。
- v2: 「今日のミッション: お金と仕事の正体を見破れ」を追加。
- 追加要素: 価値探偵ユウ、ミッションカード、探偵ノート。
- 注意: 冒険感は少しだけ。派手なゲームUIにしない。

### Page 2

- 現状: 仕事は誰かの状態を少し良くすること。
- v2: Before/After人物表現を強化。
- 追加要素: 困っている同世代キャラと、少し助かった表情。
- 注意: 困っている人を笑いものにしない。

### Page 3

- 現状: お金は価値が届いたサイン。
- v2: 「ありがとうが見える形」として、人物の表情と流れ図を組み合わせる。
- 追加要素: ありがとうカード、落ち着いたお金アイコン。
- 注意: 金額や儲けを強調しない。

### Page 4

- 現状: 価値は相手の変化。
- v2: 価値変換カードをユウとソラが並べる表現にする。
- 追加要素: 3つの変化カード、短い吹き出し。
- 注意: 文字量を増やさない。

### Page 5

- 現状: 身近な仕事を変化で見る。
- v2: ビジネス探偵ミッションとして、探偵ノート形式にする。
- 追加要素: 例カード5枚。重い場合は3枚版も作る。
- 注意: スマホゲームや動画を価値の例として扱い、依存や煽りに寄せない。

### Page 6

- 現状: 今日のまとめ。
- v2: Final Missionとして、自分の言葉で1文を書く画面にする。
- 追加要素: Final Missionカード、ユウとソラの記入シーン、ミホの短い問い。
- 注意: 模範解答ではなく、自分の言葉でよいと示す。

## 実装時の作業順

1. v2ストーリーボードとコピーを確認する。
2. HTML/CSSの既存構造を確認する。
3. 16:9版のPage 1/5/6から変更する。
4. Page 2/3/4の人物・Before/After補助を入れる。
5. 4:5版で文字と人物が詰まらないように調整する。
6. PNGを再生成する前に、HTML表示で重なりと文字量を確認する。

# Day1 Infographic v3 Design Brief

## Classification

- Audience: 子供向け
- Format: 16:9 Zoom infographic, 6 pages
- Status: v3制作準備のみ。HTML/CSS/PNGはまだ作らない。

## Why v3 Is Needed

Day1 16:9 v2 PNGは、読みやすい説明スライドにはなったが、発見ゲーム型教材画面としては弱かった。v3では、丸アイコンや名前ラベルではなく、キャラクターが行動する場面として設計する。

## Core Concept

Value Detective / 価値探偵

Day1は「働く・お金・価値」を説明する回ではなく、価値探偵ユウが「誰がどう変わるか」を観察し、子供が自分の言葉で仕事の正体を見つけるミッションとして設計する。

## Mission

お金と仕事の正体を見破れ。

## Character Actions

- ユウがミッションカードを受け取る。
- ユウが探偵ノートに観察メモを書く。
- ユウとソラが価値変換カードを机に並べる。
- ミホが正解ではなく短い問いを出す。
- Page 6では、手元でFinal Mission文を書き込む。

## Game Elements

- Mission Card
- Detective Note
- Case File
- Evidence Card
- Value Change Card
- Discovery Stamp
- Final Mission Sheet

## Visible Changes

- 困った -> 助かった
- 面倒 -> ラク
- 退屈 -> 楽しい
- わからない -> できそう

## Learner Actions

- 見る: 相手のBefore/Afterを見る。
- 選ぶ: 身近な仕事のEvidence Cardを選ぶ。
- 話す: 誰がどう変わるかを言う。
- 書く: Final Mission文を書く。

## Prohibitions

- 丸アイコンだけのキャラクター。
- 顔だけ、名前ラベルだけ。
- 「探偵」「ミッション」という語を置くだけ。
- 普通の見出し＋本文＋カードの説明スライド。
- 勝敗、採点、報酬、ランキング。
- お金の束、高級品、儲け煽り。
- 既存作品、既存キャラクター、特定漫画家の絵柄模倣。

## Acceptance Criteria

| Item | Pass |
|---|---|
| Character action | 各ページでメモ、観察、指差し、カード操作、書き込みのいずれかが見える |
| Game element | カード、ノート、ケースファイル、スタンプが学習行動と結びつく |
| Learner action | 子供が見る、選ぶ、話す、書く行動が画面からわかる |
| Change | 相手のBefore/Afterが見える |
| Safety | 起業やお金で煽らず、勝敗や採点にしない |
| Readability | Zoomで読める文字量とサイズ |

## Human Review Before Production

- v3 storyboardで、各ページのキャラクター行動が十分か。
- HTML/CSSだけで表現するか、inline SVGや画像生成素材を使うか。
- Page 5の3例にスマホゲームを残すか。
- Discovery StampやCase Fileをどこまで画面に入れるか。
- 4:5化しやすい構成か。

## 各ページの意図

### Page 1: 今日のミッション

- 意図: Day1でやることを一目で伝える。
- メインコピー: 「お金と仕事の正体を見破れ」。
- 表現: 価値探偵ユウ、探偵ノート、ミッションカードで、仕事・お金・価値を見る入口を作った。

### Page 2: 仕事とは何か

- 意図: 仕事を作業や時間ではなく、相手の状態変化として見る。
- 表現: 困った顔 -> 少し助かった顔のBefore/Afterを大きく配置した。
- 注意: Beforeを笑いものにせず、Afterも大げさな成功表現にしていない。

### Page 3: お金とは何か

- 意図: お金を価値が届いたサインとして扱う。
- 表現: 助かった人の「ありがとう」と、価値が届く -> ありがとう -> お金の小さな図解で整理した。
- 注意: お金の束、高級品、派手な儲け表現は入れていない。

### Page 4: 価値とは何か

- 意図: 価値は相手の変化で見ると伝える。
- 表現: ユウとソラが価値変換カードを並べる構成にした。
- 例: 困った -> 助かった、わからない -> できそう、退屈 -> 楽しい。

### Page 5: ビジネス探偵ミッション

- 意図: 身近な仕事を「誰がどう変わるか」で観察する。
- 表現: ビジネス探偵ノートと3枚の例カードに整理した。
- 例: コンビニ、塾/学習サポート、スマホゲーム。

### Page 6: Final Mission

- 意図: 今日の学びを自分の言葉で1文にする。
- 表現: Final Missionカードとナビゲーター・ミホの短い問いで、模範解答ではなく自分の観察を書く構成にした。

## v1から変えた点

- Page 1を「導入」から「今日のミッション」に変更。
- Page 5を5例一覧から3例の「ビジネス探偵ミッション」に変更。
- Page 6を「今日のまとめ」から「Final Mission」に変更。
- ユウ、ソラ、ミホをHTML/CSSの簡易人物表現で追加。
- 探偵ノート、ミッションカード、価値変換カードを教材独自の視覚言語として追加。
- 吹き出しは1ページ1つまでを基本にした。

## Zoom共有で読めるかの自己チェック

- 16:9の1920 x 1080px前提で、1ページ1メッセージに整理している。
- 本文は各ページ3から5行以内に抑えている。
- Page 5は3例に絞ったため、v1の5例版より画面が重くなりにくい。
- 小さい注釈は入れず、カード、ノート、吹き出しで視線を誘導している。
- 人物表現は細かい表情に頼らず、カードや言葉と組み合わせて意味が伝わるようにしている。

## Page 5を3例版にした理由

- Zoom共有では5例を横並びにすると、1枚あたりのカード幅が狭くなりやすい。
- 3例にすると、「誰が」「どう変わる」を1例ずつ短く読める。
- コンビニ、塾/学習サポート、スマホゲームは、生活の便利さ、学び、楽しさの3方向を見せられる。
- YouTuberとラーメン屋は予備例・カード例として残し、このページには詰め込まない。

## PNG再生成前に人間が確認すべき点

- Page 1の「お金と仕事の正体を見破れ」が、強すぎず入りやすいか。
- Page 2のBefore/Afterが、困っている人を笑う表現に見えないか。
- Page 3のお金表現が、金儲けや報酬で釣る印象になっていないか。
- Page 4の「わからない -> できそう」を価値変換例として採用してよいか。
- Page 5の3例にスマホゲームを残してよいか。
- Page 6のFinal Missionが、採点や成功/失敗に見えないか。
- HTML/CSSの簡易人物表現で十分か、画像生成を使うべきか。

- Page 6はFinal Missionの空欄を大きく保つ。

# Day1 16:9 v3 Prototype Visual Components

## Purpose

This note records the visual component decisions used in the Day1 v3 prototype for Page 1, Page 4, and Page 5.

## Character Components

The prototype uses inline SVG characters, not image generation and not round face icons.

Each character drawing includes:

- head
- hair
- body
- arms
- hands
- clothes
- small props
- an action pose

## Character Differentiation

- Yuu: detective notebook, pen, magnifier, pointing or observing pose.
- Sora: sticky note, card, pen, card-writing or card-pinning pose.

## Game Elements

Game elements are used as working learning tools:

- Quest Map: shows the current learning step.
- Mission Card: shows the action and final sentence for Page 1.
- Detective Note: provides the observation prompt.
- Value Change Card: lets the learner connect Before and After.
- Case File: frames Page 5 as evidence work.
- Evidence Card: records who changes and how.
- Discovery Stamp: records discovery action, not reward.

## Prototype Boundary

This component note applies only to the 3-page prototype:

- Page 1: Mission Board
- Page 4: Card Table - Value
- Page 5: Business Detective Case File

It does not approve a full 6-page v3 deck, 4:5 layout, PDF, PPTX, or script/card/worksheet propagation.

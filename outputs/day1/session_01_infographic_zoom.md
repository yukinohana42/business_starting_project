# 仕事って何？

## ただ時間を使うことではない

仕事は、誰かの状態を少し良くすること。

| もとの状態 | 変わった後 |
|---|---|
| 困った | 助かった |
| 面倒 | ラク |
| 退屈 | 楽しい |

### 問い

最近「面倒がラクになった」ことはある？

# お金って何？

## 価値が届いたサイン

お金は、いやらしい話ではない。

誰かが助かった。  
ラクになった。  
楽しくなった。

その「ありがとう」が見える形になったもの。

```text
価値が届く → ありがとう → お金
```

---

# 価値って何？

## 価値は、相手の変化

使ったあとに、相手の状態が少し良くなること。

| 変化 | 価値 |
|---|---|
| 困った → 助かった | 問題が軽くなる |
| 面倒 → ラク | 手間が減る |
| 退屈 → 楽しい | 気持ちが動く |

# 第1回 Zoom説明用 複数枚インフォグラフィック原稿

## Page 1: 導入

### タイトル

働くって何？  
お金って何？  
価値って何？

### サブタイトル

仕事は、誰かの状態を少し良くすること。

### 要点

- 今日は「仕事」「お金」「価値」を身近な例で見直す
- 起業家になる方法を覚える回ではない
- 誰かの状態がどう変わるかを見る

### 図解要素

- 今日見る3つのカード: 仕事 / お金 / 価値
- ゴールへ向かう短い道筋

### イラスト要素

- 親子で画面を見ているシンプルなイラスト
- 小さな「？」マーク

### 強調したい一言

今日は、仕事を「相手の変化」で見てみよう。

---

## Page 2: 仕事って何？

### タイトル

仕事って何？

### サブタイトル

ただ時間を使うことではない。

### 要点

- 仕事は、誰かの状態を少し良くすること
- 誰かの「困った」「面倒」「退屈」を変えること
- 物を売るだけではなく、変化を届けている

### 図解要素

- 困った / 面倒 / 退屈 のカード
- それぞれに右向き矢印

### イラスト要素

- 困っている顔
- 助かった顔
- やさしい矢印

### 強調したい一言

仕事は、誰かの状態を少し良くすること。

---

## Page 3: お金って何？

### タイトル

お金って何？

### サブタイトル

価値が届いたサイン。

### 要点

- お金は、いやらしい話ではない
- 誰かが助かった、ラクになった、楽しくなった
- その「ありがとう」が見える形になったもの

### 図解要素

```text
価値が届く → ありがとう → お金
```

### イラスト要素

- ありがとうの吹き出し
- 小さなコイン
- 価値が届く矢印

### 強調したい一言

お金は、価値が届いたサイン。

---

## Page 4: 価値って何？

### タイトル

価値って何？

### サブタイトル

価値は、相手の変化。

### 要点

- 使ったあとに、相手の状態が少し良くなる
- 価値は「すごそう」より「変わった」で見る
- 変化があるから、仕事になる

### 図解要素

- 困った → 助かった
- 面倒 → ラク
- 退屈 → 楽しい

### イラスト要素

- 3枚の変換カード
- やさしいグリーンの矢印

### 強調したい一言

価値は、相手の変化。

---

## Page 5: 身近な例

### タイトル

身近な仕事を「変化」で見る

### サブタイトル

何を売っているかより、相手がどう変わるか。

### 要点

| 仕事 | 相手の変化 |
|---|---|
| YouTuber | 退屈 → 楽しい |
| コンビニ | 今すぐ欲しい → すぐ買えた |
| ラーメン屋 | お腹が空いた → 満足 |
| 塾 | わからない → できそう |
| スマホゲーム | ひま → 夢中になる |

### 図解要素

- 仕事カード
- Before → After の矢印

### イラスト要素

- 動画、コンビニ袋、ラーメン、ノート、スマホ

### 強調したい一言

「何を売るか」より「どう変わるか」。

---

## Page 6: まとめ

### タイトル

今日のまとめ

### サブタイトル

仕事は、誰かの状態を少し良くすること。

### 要点

- 仕事とは、誰かへの価値提供
- お金とは、価値が届いたサイン
- 価値とは、相手の変化

### 図解要素

```text
困った / 面倒 / 退屈
        ↓
助かった / ラク / 楽しい
```

### イラスト要素

- 今日の1文を書くメモカード
- 小さなチェックマーク

### 強調したい一言

仕事とは、＿＿＿＿を＿＿＿＿に変えること。

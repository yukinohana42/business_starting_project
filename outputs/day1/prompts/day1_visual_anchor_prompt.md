# Day1 Visual Anchor Prompt

## Purpose

Day1「働く・お金・価値 / Value Detective」の6ページで、画風、キャラクター、教材UIの一貫性を保つためのanchor prompt。

このpromptは、各ページpromptの冒頭に必ず付ける。

## Anchor Prompt

Create a warm, original educational illustration for a Japanese junior high school grade 8 entrepreneurship workshop. The visual is not a business poster and not a normal explanation slide. It is a discovery-game learning screen where students can see what to do next.

Use a soft, friendly, organized style suitable for Zoom screen sharing. The mood is calm, curious, and lightly adventurous, with a detective-learning theme. Use warm cream paper background, soft navy text areas, muted coral accents, warm yellow discovery highlights, soft green value/help accents, and pale blue organizing lines.

Use fully original characters only. Do not imitate any existing anime, manga, game, mascot, brand, character, artist, studio, or franchise. The characters should feel like Japanese junior high school students, age-appropriate for grade 8, not preschool-like and not overly mature.

The scene should show character action, hand movements, props, and learning tools. Avoid face-only icons. Avoid characters standing without action.

Core learning idea: value is seen as a small positive change in another person. Show changes like troubled to helped, confused to able to try, bored to having fun, rushed to relieved. Do not mock the Before state and do not exaggerate the After state.

Game elements are functional learning tools, not decoration. Use Mission Card, Detective Note, Case File, Evidence Card, Value Change Card, Discovery Stamp, Reflection Stamp, Final Mission Sheet, and Question Card only when they support the learner action.

Text handling: keep generated text very minimal. Use only short English placeholder labels or simple blank areas when needed. Exact Japanese copy will be added later in layout. Leave clean open areas for overlay text and cards.

Safety: no money piles, luxury goods, winner/loser framing, scores, ranking, trophies, reward bait, business pressure, fear-based future warnings, or "you must become an entrepreneur" mood. Parent and child relationship should feel supportive.

Aspect ratio: 16:9 horizontal, designed for 1920x1080 Zoom sharing.

## Global Negative Prompt

- existing anime style, manga franchise style, game character style, mascot style
- specific artist imitation
- brand logos, school logos, copyrighted character resemblance
- money pile, cash rain, luxury car, expensive watch, gold bars
- trophy, ranking, score board, winner podium, prize
- scary warning, future anxiety, pressure to start a business
- face-only character icon
- character standing still with no learning action
- generic corporate stock art
- crowded infographic poster
- tiny unreadable text
- full Japanese paragraphs inside the generated image

## Layout Notes

- Keep a large calm area for exact Japanese overlay text.
- Leave card surfaces readable and not too small.
- Keep the main action visible at Zoom size.
- Use one main scene per page.
- Keep page-to-page consistency in character appearance and props.

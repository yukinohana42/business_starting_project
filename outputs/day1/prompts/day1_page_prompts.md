# Day1 Page Image Generation Prompts

## How To Use

For each page:

1. Prepend `outputs/day1/prompts/day1_visual_anchor_prompt.md`.
2. Add the relevant character descriptions from `outputs/day1/prompts/day1_character_prompt.md`.
3. Use the page prompt below.
4. Generate one page at a time.
5. Add exact Japanese copy later with layout tools.

Do not generate all six pages in one uncontrolled batch.

## Page 1: Mission Board

Page type: Mission Board.

Scene: Yuu stands beside a warm wooden mission board. Yuu is writing in a detective notebook while holding or receiving a mission card. The board has a large clean blank area for the Japanese headline and a separate blank Mission Card area. A small Detective Note panel shows blank lines. A Discovery Stamp placeholder is visible but subtle.

Character action: Yuu writes the observation point in the notebook and looks toward the mission card. Hands, pen, notebook, mission card, and magnifying glass must be visible.

Game elements: Mission Card, Detective Note, Discovery Stamp slot, simple Quest Map area.

Learner action shown: Confirm today's mission and the final sentence to write.

Visible change: A small strip or pair of cards suggests "work as getting money?" changing to "work as seeing someone's change." Use symbolic cards, not cash.

Text handling: Leave space for the main copy "お金と仕事の正体を見破れ。" and final sentence "仕事とは、＿＿を＿＿に変えること。"

Negative prompt additions: no cash piles, no rich lifestyle, no winner badge, no face-only Yuu.

## Page 2: Case File - Work

Page type: Case File.

Scene: A case file board shows two large panels: Before and After. In the Before panel, a student is mildly troubled by a small everyday problem. In the After panel, the same student looks slightly relieved because the situation became easier. Yuu points to both panels and compares them.

Character action: Yuu points from Before to After while holding a detective notebook. The person's change should be clear through posture and expression, not exaggerated.

Game elements: Case File, Before Card, After Card, Detective Note.

Learner action shown: Say one recent example where something became easier.

Visible change: troubled -> helped.

Text handling: Leave overlay spaces for "仕事は、誰かの状態を少し良くすること。" and a short question.

Negative prompt additions: do not laugh at the troubled person, no dramatic success pose.

## Page 3: Evidence Board - Money

Page type: Evidence Board.

Scene: A person who was helped gives a small thank-you card to another person. Yuu attaches the thank-you card to an evidence board. A small simple money sign card appears as a quiet sign of value received, not as cash or profit.

Character action: Yuu pins the thank-you evidence card onto the board. Hands and pinning motion must be visible.

Game elements: Evidence Card, Thank-you Card, Money Sign Card, Discovery Stamp.

Learner action shown: Think of something they paid for because it helped them.

Visible change: value reached someone -> thank you appears -> money is a visible sign.

Text handling: Leave space for "お金は「ありがとう」が見える形。"

Negative prompt additions: no money bundles, no shopping luxury, no profit celebration.

## Page 4: Card Table - Value

Page type: Card Table.

Scene: Yuu and Sora sit or stand around a table covered with unfinished Value Change Cards. The cards show blank Before and After spaces. Sticky notes and pens are on the table. The scene should feel like students are actively building the cards.

Character action: Yuu points at one card while Sora writes on another card or moves a sticky note. Hands, table, card edges, pen, and sticky notes must be visible.

Game elements: Value Change Card, Card Table, Discovery Stamp.

Learner action shown: Choose one card and connect Before to After.

Visible change examples: troubled -> helped, confused -> can try, bored -> fun.

Text handling: Leave overlay spaces for "価値は、相手の変化で見る。" and three short card labels.

Negative prompt additions: do not make it three static cards only; the hands and work process must be visible.

## Page 5: Business Detective Case File

Page type: Business Detective Case File.

Scene: A large case file board has three evidence cards attached with small pins and connecting lines. The three cards represent convenience store, tutoring / learning support, and smartphone game. Yuu observes with notebook and magnifying glass. Sora pins one evidence card or writes on it.

Character action: Yuu inspects the evidence board. Sora actively pins or writes an evidence card. Both characters support the investigation.

Game elements: Case File, Evidence Card, Business Detective Note, Discovery Stamp.

Learner action shown: Choose one case and say who changes and how.

Visible changes:

- rushed person -> can buy quickly and feels helped
- confused learner -> feels able to try
- bored time -> fun time

Text handling: Leave space for "この仕事は、誰をどう変えている？" and short overlay labels. Exact Japanese card text will be added later.

Negative prompt additions: no YouTuber or ramen shop on this main page, no crowded five-example layout, no money hype.

## Page 6: Final Mission Sheet

Page type: Final Mission Sheet.

Scene: A large final mission worksheet lies on a table. Yuu or Sora writes a sentence into a large blank line. Miho gently offers a Question Card, not an answer card. The mood is reflective and calm.

Character action: A student hand writes on the Final Mission Sheet. Miho holds a short question card nearby. Yuu or Sora reviews the notes.

Game elements: Final Mission Sheet, Reflection Stamp, Question Card, Detective Note.

Learner action shown: Write one sentence in their own words.

Visible change: seeing work vaguely -> seeing who changes and how.

Text handling: Leave a large clean blank for "仕事とは、＿＿を＿＿に変えること。"

Negative prompt additions: no success/failure mark, no score, no grade, no prize.

## Page-Level Review Checklist

For each generated page, check:

- Does it clearly match the page type?
- Is at least one character doing a learning action?
- Are hands, props, and gaze visible?
- Does the game element support the learner action?
- Is the value or learning change visible?
- Is the image not overloaded with text?
- Is it safe: no pressure, no money hype, no competition, no IP mimicry?

# Day1 Infographic v3 Storyboard

## Global Rule

各ページは説明スライドではなく、発見ゲーム型の教材画面にする。キャラクターは丸アイコンではなく、行動している人物として描く。

## Page 1: Mission Board

- Message: お金と仕事の正体を見破れ。
- Character action: ユウがミッションカードを受け取り、探偵ノートに「見るポイント: 誰がどう変わる？」と書く。
- Game element: Mission Card / Detective Note / Discovery Stamp枠。
- Learner action: 今日のミッションと最後に書く文を確認する。
- Visible change: 「仕事=お金をもらう？」から「仕事=誰かの変化を見る」へ。
- Required visual: 手元のノート、ペン、ミッションカード。
- Prohibited v2 pattern: ユウの丸アイコンを横に置くだけ。

## Page 2: Case File - Work

- Message: 仕事は、誰かの状態を少し良くすること。
- Character action: ユウがBefore/Afterの2コマを指差して比べる。
- Game element: Case File / Before Card / After Card.
- Learner action: 最近「面倒がラクになった」例を1つ話す。
- Visible change: 困っている同世代の子 -> 少し助かった同世代の子。
- Required visual: 状況がわかる小物、表情、姿勢。
- Prohibited v2 pattern: 顔アイコン2つと矢印だけ。

## Page 3: Evidence Board - Money

- Message: お金は「ありがとう」が見える形。
- Character action: 助かった人がありがとうカードを渡し、ユウがそれをEvidence Boardへ貼る。
- Game element: Evidence Card / Thank-you Card / Money Sign Card.
- Learner action: お金を払ってよかったものを、変化で言う。
- Visible change: 価値が届く -> ありがとう -> お金のサイン。
- Required visual: カードを渡す手元、貼る動作。
- Prohibited v2 pattern: 3ステップ図解だけ。

## Page 4: Card Table - Value

- Message: 価値は、相手の変化で見る。
- Character action: ユウとソラが机の上で未完成の価値変換カードを並べ、空欄を埋める。
- Game element: Value Change Card / Card Table / Discovery Stamp.
- Learner action: もとの状態と変わった後を組み合わせる。
- Visible change: 困った -> 助かった、わからない -> できそう、退屈 -> 楽しい。
- Required visual: 机、手元、未完成カード、付箋。
- Prohibited v2 pattern: 3枚カードがただ並んでいるだけ。

## Page 5: Business Detective Case File

- Message: この仕事は、誰をどう変えている？
- Character action: ユウとソラがCase Fileに3つのEvidence Cardを貼り、「誰が」「どう変わる」を書く。
- Game element: Business Detective Note / Evidence Card / Case File / Discovery Stamp.
- Learner action: コンビニ、塾/学習サポート、スマホゲームから1つ選び、誰がどう変わるかを言う。
- Visible change:
  - コンビニ: 急いでいる人 -> すぐ買えて助かる
  - 塾/学習サポート: わからない人 -> できそうになる
  - スマホゲーム: 退屈な時間 -> 楽しい時間になる
- Required visual: Case Fileの見出し、Evidence Card、メモ欄、発見スタンプ。
- Prohibited v2 pattern: 普通の3例カード。

## Page 6: Final Mission Sheet

- Message: 仕事の正体を、自分の言葉で言え。
- Character action: ユウまたはソラがFinal Mission Sheetへ書き込み、ミホが問いカードを差し出す。
- Game element: Final Mission Sheet / Reflection Stamp / Question Card.
- Learner action: 「仕事とは、＿＿を＿＿に変えること。」を書く。
- Visible change: なんとなく仕事を見る -> 誰がどう変わるかで見る。
- Required visual: 書き込む手元、大きな空欄、ミホの問いカード。
- Prohibited v2 pattern: 空欄カードと顔アイコンだけ。

## Production Stop Check

HTML/CSSへ進む前に、以下がすべてYesであること。

- 各ページにキャラクター行動がある。
- 各ページのゲーム要素が学習行動と結びついている。
- 子供が何をするかが画面からわかる。
- 相手の変化が見える。
- 丸アイコンだけではない表現方法が決まっている。

# 第1回インフォグラフィック v2 イラストプロンプト案

この文書は、将来のイラスト作成やHTML/CSS反映に向けた設計案です。今回はPNGや画像を再生成しません。

## 共通方針

- 完全オリジナルキャラクター。
- 既存漫画、既存アニメ、既存ゲーム、特定漫画家の絵柄を模倣しない。
- 中学2年生が感情移入しやすい、落ち着いた親しみのある人物。
- 親が見ても安心できる服装、表情、言葉。
- みんほす！らしい温かい色、やわらかい線、読みやすい構図。
- 派手なネオン、バトル演出、勝利ポーズ、ランキング表現は使わない。

## キャラクター設定

### 価値探偵ユウ

- 中学生。
- 好奇心が強い。
- 観察メモや探偵ノートを持つ。
- 表情は少しワクワクしている。
- 役割は「誰がどう変わるか」を見つけること。

### アイデアビルダー・ソラ

- 中学生。
- ラフスケッチやカードを使って考える。
- 表情は前向きで、試してみようという雰囲気。
- 役割は「考えを形にすること」。

### ナビゲーター・ミホ

- 年上すぎない親しみやすい案内役。
- 先生っぽくなりすぎない。
- 表情は落ち着いていて、問いを出す。
- 役割は「困ったときにヒントを出すこと」。

## 画像生成前チェック

- 既存作品名、既存キャラクター名、特定漫画家名が入っていないか。
- キャラが完全オリジナルか。
- 親が見ても安心できるか。
- 価値が人物の変化として見えるか。
- 文字を入れる余白が残っているか。
- 16:9と4:5の両方に転用できるか。

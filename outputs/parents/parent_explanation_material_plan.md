# Parent Explanation Material Plan

## Purpose

親が家庭内起業ワークショップの意味を理解し、先生・採点者・ゲームマスターではなく、問いを出す伴走者として関われるようにする。

この資料は、起業を押し付けるための説明ではない。子供が「価値を見つける」「相手の変化を見る」「小さく試す」「反応から改善する」経験を安全に支えるための親向け案内である。

## Target Reader

- 中学2年生の子供と家庭内またはZoomでワークショップを実施する親。
- 起業教育に詳しくなくても使えること。
- 子供の答えを評価しすぎたり、正解を先に言ったりしがちな親にも安全に使えること。

## Use Case

- Day1開始前の親向け説明。
- Day1からDay5までの全体像の共有。
- 各回の前に親が声かけを確認する。
- 実施中に親が迷ったときのFacilitator Guide。
- 実施後のふりかえり。

## Recommended Format

- 16:9 親説明スライド原稿。
- PDF化可能なMarkdownまたはスライド原稿。
- 子供向けほどゲームUIを強くしない。
- 温かいが落ち着いたデザイン。
- 図解、問いカード、OK/NG Response Sheet、Safety Boundary Cardを中心にする。

## Recommended Page Count

6から8ページ。

最小版は6ページ:

1. このワークショップで大事にすること。
2. 親の役割。
3. Day1からDay5の全体像。
4. 親の問いかけ例。
5. OK/NG声かけ。
6. 安全境界と次の進め方。

## Page Structure Draft

| Page | Material Type | Message | Visual Direction |
| --- | --- | --- | --- |
| 1 | Parent Briefing | 起業を押し付けず、価値を見る練習をする | 親子が同じ机で資料を見る落ち着いた場面 |
| 2 | Facilitator Guide | 親は先生でも採点者でもなく伴走者 | 親が問いカードを横から差し出す |
| 3 | Session Flow Map | Day1からDay5は価値発見から小さな実験、発表へ進む | 5回の流れを穏やかなマップで見せる |
| 4 | Question Card | 正解より問いを出す | 「誰がどう変わる？」などの問いカード |
| 5 | OK/NG Response Sheet | 評価ではなく観察・問い・試行・改善を支える | OK/NGの2列、言い換え例 |
| 6 | Safety Boundary Card | 無理な販売、比較、金儲け煽りを避ける | 安全境界カードと家庭内実施の範囲 |
| 7 | Praise Guide | 結果ではなくプロセスを褒める | 観察、問い、試行、改善の小さな記録 |
| 8 | Reflection Support Sheet | 実施後に次へつなげる | 親子で短く振り返るメモ欄 |

## Messages To Parents

- 起業するかどうかを決める教材ではない。
- 価値を「相手の変化」として見る練習である。
- お金は煽る材料ではなく、価値が届いたサインとして扱う。
- ゲーム要素は報酬や勝敗ではなく、学びの足場である。
- 親は正解を教える人ではなく、問いを出す伴走者である。
- 子供の答えを評価するより、観察、問い、試行、改善を支える。

## OK / NG For Parents

OK:

- 「誰がどう変わる？」
- 「今はどうしている？」
- 「それが変わると何がうれしい？」
- 「小さく試すなら何ができそう？」
- 「反応が薄かったら、どこを変えてみる？」
- 「自分の言葉で言うと？」

NG:

- 「それは儲かるの？」
- 「それは起業にならないよ」
- 「正解はこうだよ」
- 「すごい/すごくない」で評価する
- 兄弟や他の子と比べる
- 発表を採点会にする

## Day1-Day5 Overview

| Day | Theme | Parent Support |
| --- | --- | --- |
| Day1 | 働く・お金・価値 | 誰がどう変わるかを問い返す |
| Day2 | 困りごとの見つけ方 | 日常の観察を聞く。無理なインタビューをさせない |
| Day3 | アイデア作り | すごさではなく、誰の何を助けるかを見る |
| Day4 | 小さな実験 | 赤信号を失敗扱いせず、次の改善につなげる |
| Day5 | 価値を伝える | 発表のうまさではなく、相手の変化が伝わったかを見る |

## Praise Guide

褒める対象:

- 観察したこと。
- 問いを作ったこと。
- 相手を具体的にしたこと。
- 小さく試そうとしたこと。
- 反応を見たこと。
- 改善を考えたこと。
- 自分の言葉でまとめたこと。

避ける対象:

- 儲かりそう。
- 一番すごい。
- 起業家っぽい。
- 他の子より良い。
- 親の好みに近い。

## Image Generation Or Diagram

親向け資料では、文字の正確さと実用性が重要です。全ページを画像生成に任せません。

画像生成に任せる:

- 親子の距離感。
- 温かいFacilitator Guideの場面。
- 問いカードや安全境界カードの雰囲気。
- 親が横から支える構図。

後編集・図解・Markdown/スライド組版で補う:

- 正確な日本語本文。
- OK/NG声かけ例。
- Day1-Day5の表。
- チェックリスト。
- 安全境界。
- 余白と印刷可読性。

## Human Review Before Production

- 親が先生、採点者、ゲームマスターに見えないか。
- 起業を押し付けていないか。
- お金で煽っていないか。
- OK/NG声かけが実際に使えるか。
- 子供向けゲームUIを強くしすぎていないか。
- 画像生成に任せる部分と後編集で載せる部分が分かれているか。

# Parent Material Visual Design Brief v2

## Classification

- Audience: 親向け
- Format: Parent briefing, checklist, question cards, facilitator guide
- Status: 制作準備のみ。HTML/CSS/PDF/PPTXはまだ作らない。

## Screen Concept

Parent Facilitator Guide / Question Card / OK-NG Response Sheet / Safety Boundary

## Purpose

親が先生、採点者、ゲームマスターにならず、問いを出す伴走者として関われるようにする。

## Parent Actions

- 先に正解を言わない。
- 子供の観察を聞く。
- 誰がどう変わるかを問い返す。
- 安全な範囲を守る。
- 結果ではなく過程を褒める。
- 反応が悪いときに改善へつなげる。

## Materials and Game-Adjusted Elements

- Facilitator Map
- Question Card
- OK/NG Voice Card
- Safety Boundary Card
- Praise Guide
- Reflection Support Sheet

## Parent Prompts

- 誰がどう変わる？
- 今はどうしている？
- 小さく試すなら？
- 反応が悪かったら、どこを変える？
- きれいな答えじゃなくて、自分の言葉なら？

## Praise Targets

- 観察したこと。
- 問いを作ったこと。
- 試したこと。
- 改善を考えたこと。
- 自分の言葉でまとめたこと。

## Prohibitions

- 起業しないと困ると煽る。
- 稼げそうかで評価する。
- すごい/すごくないで評価する。
- 親の好みの答えに誘導する。
- 兄弟や他人と比べる。
- 発表を採点する。
- 親をゲームマスターとして扱う。

## Acceptance Criteria

| Item | Pass |
|---|---|
| Parent role | ファシリテーターとして描かれている |
| Voice | 問い、観察、伴走が中心 |
| Practicality | OK/NG例、問いカード、安全境界がある |
| Safety | 起業、お金、競争で煽らない |
| Design | 落ち着いたが温かく、無機質ではない |

## Human Review Before Production

- 親がそのまま使える問いになっているか。
- NG声かけが具体的か。
- 子供向けゲームUIを強くしすぎていないか。
- 親が採点者に見えないか。

# Parent Facilitator Visual Design Brief

## Audience

親向け説明資料、親チェックリスト、親用問いカード、家庭内/Zoom進行補助。

## Mission

親が評価者ではなく、問いを出す伴走者として関われるようにする。

## Learning Goal

親が「正解を教える」から「観察を助ける」に切り替えられる。

## Main Page Types

- Parent Facilitator Guide
- Session Flow Map
- Question Card
- OK/NG Response Sheet
- Safety Boundary Card
- Observation Log
- Praise Guide
- Reflection Support Sheet

## Game Elements

親向けではゲーム感を弱め、実用的な支援ツールとして扱う。

- Question Card
- OK/NG Voice Card
- Safety Boundary Card
- Observation Log
- Reflection Support Sheet
- Praise Guide

## Parent Actions

- 先に正解を言わない。
- 子供の観察を聞く。
- 「誰がどう変わる？」と問い返す。
- 安全な範囲を守る。
- 結果ではなく、観察、問い、試行、改善を褒める。
- 反応が薄いときは改善へつなげる。

## Visible Changes

- 親が教える -> 親が問いを出す
- 親が採点する -> 親が観察を支える
- すごさを褒める -> プロセスを褒める
- 不安で押す -> 安全に小さく試す

## Image Generation Notes

Generate calm, warm facilitator scenes. Show a parent sitting beside, not above, the child. The parent holds a Question Card or OK/NG Voice Card. The child writes or talks while the parent listens. Use practical guide visuals rather than game-heavy UI.

Avoid teacher-at-blackboard, judge panel, scoring sheet, pressure, fear warnings, money hype, and parent pointing down at the child.

Exact Japanese OK/NG phrases should be added later.

## Suggested Material Sequence

| Material | Type | Visual Core | Parent Action |
| --- | --- | --- | --- |
| 1 | Parent Facilitator Guide | Parent beside child with question cards | Understand role |
| 2 | Session Flow Map | 45-minute home/Zoom flow | Keep pace |
| 3 | Question Card | Short reusable questions | Ask, not answer |
| 4 | OK/NG Response Sheet | Two columns of voice examples | Avoid evaluation |
| 5 | Safety Boundary Card | Safe interview/test boundaries | Protect safety |
| 6 | Reflection Support Sheet | Parent helps child summarize | Support reflection |

## Negative Prompt

- parent as judge
- teacher lecturing
- scoring sheet
- "correct answer" mood
- comparing siblings
- fear about future
- pressure to become entrepreneur
- money reward
- childish game UI

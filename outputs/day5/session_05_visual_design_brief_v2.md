# Day5 Visual Design Brief v2

## Classification

- Audience: 子供向け
- Session: Day5 Pitch Challenger / ピッチチャレンジャー
- Status: 制作準備のみ。HTML/CSS/PNGはまだ作らない。

## Screen Concept

Pitch Stage / Story Board / Listener Memo

## Mission

3分で価値の変化を伝えろ。

## Character Actions

- ユウが価値の変化を確認する。
- ソラがProblem / Customer / Change / Testのカードを並べる。
- ミホが聞き手メモカードを持つ。
- 子供キャラが発表し、聞き手の反応を見て改善点を選ぶ。

## Game Elements

- Pitch Card
- Value Story Board
- Listener Memo Card
- Final Quest Map
- Reflection Stamp
- Next Challenge Card

## Visible Change

- 何をするかわからない -> 誰の何をどう良くするかが伝わる
- アイデアだけ -> 価値の変化として伝わる
- 発表して終わり -> 聞いた反応で改善する

## Learner Actions

- 価値の変化を1文にする。
- 問題、相手、変化、試し方を並べる。
- 3分で話す。
- 聞き手メモから改善点を1つ選ぶ。

## Prohibitions

- 発表の勝ち負け。
- 親の採点会。
- 起業家っぽさの押し付け。
- うまく話せたかだけを評価する。
- 普通のピッチ説明スライドにする。

## Acceptance Criteria

- Pitch Stageとして見える。
- 聞き手メモが採点ではなく反応記録になっている。
- 価値の変化が発表の中心にある。
- 改善点を選ぶ流れがある。

## Human Review Before Production

- 親の役割が審査員に見えないか。
- 発表が勝敗に見えないか。
- 子供が緊張しすぎない設計か。

# Day5 Visual Design Brief

## Session

Day5: Pitch Challenger / ピッチチャレンジャー

## Mission

3分で価値の変化を伝えろ。

## Learning Goal

すごさではなく、誰の何をどう良くするかが伝わるように話せる。

## Main Page Types

- Mission Board
- Pitch Stage
- Value Story Board
- Listener Memo
- Reflection Board
- Final Mission Sheet

## Game Elements

- Pitch Card
- Value Story Board
- Listener Memo Card
- Final Quest Map
- Reflection Stamp
- Next Challenge Card

## Character Roles

- Yuu checks whether the value change is clear.
- Sora arranges Problem / Customer / Change / Test cards.
- Miho offers Listener Memo and reminds adults not to judge or score.

## Learner Actions

- 書く: 価値の変化を1文にする。
- 並べる: 問題、相手、変化、試し方を並べる。
- 話す: 3分で伝える。
- 見る: 聞き手メモの反応を見る。
- 変える: 次に直す点を1つ選ぶ。

## Visible Changes

- 何をするかわからない -> 誰の何をどう良くするかが伝わる
- アイデアだけ -> 価値の変化として伝わる
- 発表して終わり -> 聞いた反応で改善する

## Image Generation Notes

Generate a supportive pitch practice scene, not a competition stage. The learner is presenting near a Value Story Board. Listener Memo should look like reaction notes, not a score sheet. Miho should guide adults to ask questions, not judge.

Avoid spotlights, trophies, winner/loser framing, ranking boards, business-contest mood, and adult judges.

Exact Japanese script and pitch card copy should be added after image generation.

## Suggested Page Sequence

| Page | Type | Visual Core | Learner Action |
| --- | --- | --- | --- |
| 1 | Mission Board | Final quest map and pitch mission | Confirm mission |
| 2 | Value Story Board | Problem/Customer/Change/Test cards | Arrange story |
| 3 | Pitch Stage | Learner presents calmly | Speak |
| 4 | Listener Memo | Listener writes reaction, not score | Look at reaction |
| 5 | Reflection Board | One improvement is selected | Choose next change |
| 6 | Final Mission Sheet | "私たちのアイデアは、＿＿を＿＿に変える。" | Summarize |

## Negative Prompt

- pitch contest
- judges scoring
- trophy
- winner/loser
- business pressure
- luxury startup imagery
- face-only characters
- generic pitch deck slide

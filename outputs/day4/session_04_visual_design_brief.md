# Day4 Visual Design Brief

## Session

Day4: Experiment Designer / 実験デザイナー

## Mission

本当に欲しい人がいるか小さく試せ。

## Learning Goal

PMFを「作ったものと欲しい人の気持ちがピタッとはまること」と理解し、小さな実験で反応を見る。

## Main Page Types

- Mission Board
- Mini Experiment Lab
- Signal Board
- Reaction Log
- Red Signal Improvement Card
- Final Mission Sheet

## Game Elements

- Experiment Card
- PMF Signal Card
- Reaction Log
- Red Signal Improvement Card
- Mini Test Map
- Learning Stamp

## Character Roles

- Sora chooses a small Experiment Card.
- Miho shows Safety Boundary Card and keeps the experiment safe.
- Yuu or a learner character records reactions in the Reaction Log.

## Learner Actions

- 選ぶ: 小さく試す方法を選ぶ。
- 書く: 何を見たら欲しいとわかるかを書く。
- 比べる: 青/黄/赤信号を選ぶ。
- 試す: 家庭内で安全な小さな実験を考える。
- 変える: 赤信号を改善メモに変える。

## Visible Changes

- 欲しいかわからない -> 反応が見える
- 作りっぱなし -> 試して改善する
- 赤信号 -> 次の改善ヒント
- 思い込み -> 相手の反応

## Image Generation Notes

Generate a warm mini lab scene with cards, not a science lab parody. Show Sora selecting a simple experiment card, Miho holding a safety boundary card, and a reaction log with blue/yellow/red signal cards.

Do not show risky experiments, selling to strangers, pressure to earn money, or a failure mark. Red signal should feel like useful feedback.

Exact Japanese PMF explanation and log labels should be added later.

## Suggested Page Sequence

| Page | Type | Visual Core | Learner Action |
| --- | --- | --- | --- |
| 1 | Mission Board | Sora receives experiment mission | Confirm mission |
| 2 | Experiment Lab | Small test cards on a table | Choose a small test |
| 3 | Signal Board | Blue/yellow/red cards with calm meaning | Choose what reaction means |
| 4 | Reaction Log | A learner writes what happened | Record reaction |
| 5 | Improvement Card | Red signal turns into next change | Decide one change |
| 6 | Final Mission Sheet | "最初に試す一番小さいことは、＿＿。" | Write next test |

## Negative Prompt

- red signal as failure or punishment
- forced selling
- scary warning
- judge or score
- expensive business equipment
- face-only icons
- generic PMF explanation slide

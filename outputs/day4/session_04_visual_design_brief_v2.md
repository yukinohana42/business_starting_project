# Day4 Visual Design Brief v2

## Classification

- Audience: 子供向け
- Session: Day4 Experiment Designer / 実験デザイナー
- Status: 制作準備のみ。HTML/CSS/PNGはまだ作らない。

## Screen Concept

Mini Experiment Lab / Signal Board / Reaction Log

## Mission

本当に欲しい人がいるか小さく試せ。

## Character Actions

- ソラが小さな実験カードを選ぶ。
- ミホが安全境界カードを示す。
- 子供キャラが反応ログに青/黄/赤信号を書き込む。
- 赤信号カードを改善メモに変える。

## Game Elements

- Experiment Card
- PMF Signal Card
- Reaction Log
- Red Signal Improvement Card
- Mini Test Map
- Learning Stamp

## Visible Change

- 欲しいかわからない -> 反応が見える
- 作りっぱなし -> 試して改善する
- 赤信号 -> 次の改善ヒント
- 思い込み -> 相手の反応

## Learner Actions

- 小さく試す方法を選ぶ。
- 何を見たら欲しいとわかるかを決める。
- 反応を青/黄/赤で記録する。
- 次に変えることを書く。

## Prohibitions

- 赤信号を失敗扱いする。
- 無理に販売させる。
- 完璧な完成品を求める。
- 親が評価者になる。
- 普通のPMF説明スライドにする。

## Acceptance Criteria

- Experiment Labとして見える。
- 実験カードとReaction Logが学習行動を進めている。
- 赤信号が改善メモへ変わる。
- 安全境界が見える。

## Human Review Before Production

- 実験例が家庭内で安全か。
- PMFを難しい言葉で扱いすぎていないか。
- 赤信号の扱いが安心できるか。

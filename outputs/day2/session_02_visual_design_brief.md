# Day2 Visual Design Brief

## Session

Day2: Problem Hunter / 困りごとハンター

## Mission

日常に隠れた価値の種を見つけろ。

## Learning Goal

不満、面倒、好き、もっと楽しくしたい気持ちを、価値の種として見つけられる。

## Main Page Types

- Mission Board
- Daily Observation Map
- Evidence Board
- Question Card
- Problem Hunter Log
- Final Mission Sheet

## Game Elements

- Problem Card
- Observation Map
- Question Card
- Interview Safety Card
- Discovery Stamp
- Problem Hunter Log

## Character Roles

- Yuu observes daily scenes and writes sticky notes.
- Miho offers safe question cards and reminds the learner not to force interviews.
- A student character writes one small problem on a Problem Card.

## Learner Actions

- 見る: 家、学校、部活、スマホ時間の場面を見る。
- 選ぶ: 気になる場面を1つ選ぶ。
- 書く: 困りごとをProblem Cardに書く。
- 話す: 「いつ困る？」「今はどうしている？」を声に出す。
- 試す: 安全な範囲で聞ける相手を決める。

## Visible Changes

- 不満 -> 気づき
- 面倒 -> 解決の種
- 好き -> もっと欲しい
- なんとなく困る -> 具体的に言える

## Image Generation Notes

Generate scenes where Yuu and Miho are active, not standing as icons. Use an observation map as a real board with sticky notes. Show hands placing sticky notes, circling a daily scene, or choosing a Question Card.

Keep generated text minimal. Exact Japanese questions and Problem Card copy should be added later.

Avoid showing unsafe interviews, strangers online, personal data, or laughing at people who are struggling.

## Suggested Page Sequence

| Page | Type | Visual Core | Learner Action |
| --- | --- | --- | --- |
| 1 | Mission Board | Yuu finds hidden value seeds on a daily map | Confirm the mission |
| 2 | Daily Observation Map | Home/school/club/smartphone scenes with sticky notes | Choose a scene |
| 3 | Problem Card Table | A student writes one problem card | Write one problem |
| 4 | Question Card | Miho offers safe questions | Pick one question |
| 5 | Interview Safety Board | Safe people and unsafe boundaries are visible | Choose a safe person |
| 6 | Final Mission Sheet | Fill "価値の種は、＿＿の中にある。" | Summarize |

## Negative Prompt

- teasing people who are troubled
- personal information collection
- interviewing strangers online
- ranking problems
- business pressure
- face-only character icons
- generic explanation slide

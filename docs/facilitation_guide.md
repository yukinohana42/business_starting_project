# 進行ガイド

## 基本姿勢

親は話しすぎません。子供が考える時間、書く時間、選ぶ時間、話す時間を作ります。

目安は、親が話す時間3割、子供が動く時間7割です。

## 進め方の型

1. 今日の問いを出す
2. 身近な例を1つだけ出す
3. 子供に考えさせる
4. カードやワークシートで形にする
5. なぜそう思ったかを聞く
6. まとめを1文で書く

## 深掘り質問

- それは誰が困っている？
- どれくらい困っている？
- いつ困っている？
- 今はどうしている？
- その方法の何が面倒？
- それを使うと、どんな気持ちになる？
- 小さく試すなら、明日できることは何？

## 困ったとき

子供が黙った場合:

- 選択肢カードを出す
- 自分の話ではなく、友達や親の例に変える
- いったん書かずに話すだけにする

話が脱線した場合:

- 「それは誰の何を良くしている？」に戻す

アイデアが大きすぎる場合:

- 「明日、家で試すなら何にする？」に戻す

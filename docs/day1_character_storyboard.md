# Day1 キャラクター入りインフォグラフィック v2 ストーリーボード

Day1の既存6枚構成を保ちつつ、同世代オリジナルキャラクター、ミッション化、Before/After表現を加えるための設計です。PNG再生成はこの文書では行いません。

## Page 1: 今日のミッション

- ページタイトル: 働くって何？ お金って何？ 価値って何？
- ミッション名: お金と仕事の正体を見破れ
- 伝える概念: 仕事は相手の変化として見る。
- 登場キャラクター: 価値探偵ユウ。
- 人物イラストの内容: ユウが探偵ノートとミッションカードを持ち、3つのキーワードを見る。
- Before/Afterの変化: 「仕事=お金をもらうこと？」から「仕事=誰かの状態を良くすること」へ。
- 図解要素: 仕事、価値、お金の3カードと、今日のゴール文。
- 吹き出し案: 「今日は、相手がどう変わるかを見てみよう」
- 親が見ても安心できる表現: 起業家になる回ではなく、見方を学ぶ回だと示す。
- 文字量の目安: 既存Page 1より1行増やす程度。説明は台本に置く。
- 次ページへのつながり: まず「仕事」を相手の変化で見る。

## Page 2: 仕事って何？

- ページタイトル: 仕事は、誰かの状態を少し良くすること
- ミッション名: 困った顔を助かった顔に変えろ
- 伝える概念: 仕事は時間や作業ではなく、相手の状態を良くすること。
- 登場キャラクター: ユウ、困っている同世代キャラクター。
- 人物イラストの内容: 忘れ物で困る子、助けてもらって安心する子。
- Before/Afterの変化: 困った -> 助かった。
- 図解要素: Before/Afterカード、矢印、変化ラベル。
- 吹き出し案: 「何をしたかより、誰が助かったか」
- 親が見ても安心できる表現: 困っている子を笑わない。少し良くなる変化として見せる。
- 文字量の目安: 見出し、短い説明、問い1つ。
- 次ページへのつながり: 変化が届いたとき、お金はどう見えるかに進む。

## Page 3: お金って何？

- ページタイトル: お金は、価値が届いたサイン
- ミッション名: ありがとうの正体を見つけろ
- 伝える概念: お金は金儲けではなく、価値が届いたサイン。
- 登場キャラクター: ユウ、助かった人。
- 人物イラストの内容: 困りごとが軽くなった人が「ありがとう」と言い、カードが「お金」のアイコンへつながる。
- Before/Afterの変化: 助かった気持ち -> ありがとう -> お金。
- 図解要素: 価値が届く、ありがとう、お金の3ステップ。
- 吹き出し案: 「お金って、ありがとうが見える形なんだ」
- 親が見ても安心できる表現: お金を煽らず、感謝と価値の流れとして扱う。
- 文字量の目安: 3ステップの短文のみ。
- 次ページへのつながり: では、価値そのものは何かを見にいく。

## Page 4: 価値って何？

- ページタイトル: 価値は、相手の変化
- ミッション名: 変化カードを完成させろ
- 伝える概念: 価値は商品名ではなく、使った後の状態変化。
- 登場キャラクター: ユウ、アイデアビルダー・ソラ。
- 人物イラストの内容: 2人が「困った -> 助かった」「面倒 -> ラク」「退屈 -> 楽しい」のカードを並べる。
- Before/Afterの変化: 状態カード3組。
- 図解要素: 価値変換カード、矢印、短い補足。
- 吹き出し案: 「すごそうより、変わったかを見る」
- 親が見ても安心できる表現: 正解当てではなく、変化を見つけるワークとして見せる。
- 文字量の目安: 3カード中心。本文は最小限。
- 次ページへのつながり: 身近な仕事も変化で見てみる。

## Page 5: ビジネス探偵ミッション

- ページタイトル: 身近な仕事を「変化」で見る
- ミッション名: ビジネス探偵ミッション
- 伝える概念: 何を売っているかより、相手がどう変わるか。
- 登場キャラクター: ユウ、ソラ。
- 人物イラストの内容: 2人が探偵ノートに、コンビニ、ラーメン屋、塾、動画、スマホゲームの変化をメモする。
- Before/Afterの変化: 退屈 -> 楽しい、今すぐ欲しい -> すぐ買えた、わからない -> できそう。
- 図解要素: ビジネス探偵カード5枚。必要なら3枚に減らせる。
- 吹き出し案: 「この仕事は、誰をどう変えてる？」
- 親が見ても安心できる表現: スマホゲームや動画は、価値の例として扱い、依存や煽りに寄せない。
- 文字量の目安: 例カード1枚あたり1行。画面が重い場合は3例版を作る。
- 次ページへのつながり: 見つけた変化を、自分の言葉でまとめる。

## Page 6: Final Mission

- ページタイトル: 今日のFinal Mission
- ミッション名: 仕事の正体を1文で言え
- 伝える概念: 学びを自分の言葉でまとめる。
- 登場キャラクター: ユウ、ソラ、ナビゲーター・ミホ。
- 人物イラストの内容: ユウとソラがワークシートに1文を書き、ミホが問いを添える。
- Before/Afterの変化: 「なんとなく仕事を見る」から「誰がどう変わるかで見る」へ。
- 図解要素: Final Missionカード、空欄、困った/面倒/退屈 -> 助かった/ラク/楽しい。
- 吹き出し案: 「仕事とは、＿＿を＿＿に変えること」
- 親が見ても安心できる表現: 模範解答ではなく、自分の言葉で書ければよいと示す。
- 文字量の目安: 既存Page 6と同程度。空欄を大きく残す。
- 次ページへのつながり: Day2では、日常の困りごとから価値の種を探す。

## v2制作時の注意

- PNG再生成前にこのストーリーボードを確認する。
- 既存のHTML/CSS構造を活かし、文字量を増やしすぎない。
- 16:9と4:5の両方で、人物が小さくなりすぎないようにする。
- 既存作品や特定漫画家の絵柄を指定しない。
- ゲーム的な演出より、発見と問いを主役にする。

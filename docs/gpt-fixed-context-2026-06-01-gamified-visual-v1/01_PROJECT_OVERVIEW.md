# Project Overview

## 一言説明

中学2年生の子供たちが起業セミナーに参加する前に、家庭で5回の準備ワークショップを行い、「誰かにとって価値あることを見つけ、小さく試す力」を育てるプロジェクトです。

## 対象者

- 中学2年生の男の子2人
- 起業やビジネスの前提知識はほぼない
- 座学だけだと飽きやすい
- ゲーム、部活、スマホ、勉強、友達、親、文化祭、お店など身近な題材が入りやすい
- 進行役は主に親

## 実施形式

- 家庭内ワークショップ、またはZoom画面共有で実施します。
- 1回30から60分を想定します。
- 親が話す時間より、子供が考える、話す、書く、選ぶ時間を多くします。
- 親は先生ではなく、問いを出すファシリテーターとして関わります。

## 目指す力

- 観察する: 困りごと、面倒、好き、欲しい気持ちを見つける。
- 聞く: 思い込みではなく、相手の話から考える。
- 試す: 完璧に考えず、小さく動いて反応を見る。
- 伝える: 誰の何をどう良くするかを短く言う。

## 5回構成

1. Day1: 働く・お金・価値 / Value Detective
2. Day2: 価値の見つけ方 / Problem Hunter
3. Day3: アイデア作り / Idea Builder
4. Day4: PMFと小さな実験 / Experiment Designer
5. Day5: 3分ピッチ / Pitch Challenger

## 4回圧縮版

4回に圧縮する場合は、Day3のアイデア作りとDay4のPMF・小さな実験を統合します。圧縮版でも、「誰の何をどう良くするか」と「小さく試して反応を見ること」は削らないでください。

## 成果物の種類

- セッション別README、進行台本、スライド原稿、カード、ワークシート、振り返り。
- 親向け説明資料。
- Zoom共有用インフォグラフィック。
- A4印刷用カード、ワークシート。
- AI/Codex用プロンプトとSkill定義。

## 今回固定した追加方針

- 各回をミッション構造にする。
- 同世代オリジナルキャラクターを使い、抽象概念を人物の変化として見せる。
- マンガ的1コマ表現は理解補助として使う。
- バッジ、スタンプ、進捗は評価ではなく学びの記録にする。
- Day1インフォグラフィックv2では、Page 1を今日のミッション、Page 5をビジネス探偵ミッション、Page 6をFinal Missionとして再設計する。

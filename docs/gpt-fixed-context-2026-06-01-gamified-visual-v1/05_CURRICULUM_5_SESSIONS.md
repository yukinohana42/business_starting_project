# Curriculum: 5 Sessions

## 全体像

| 回 | テーマ | ミッション | ゴール |
|---|---|---|---|
| 1 | 働く・お金・価値 | Value Detective / 価値探偵 | 仕事は価値提供だとわかる |
| 2 | 価値の見つけ方 | Problem Hunter / 困りごとハンター | 不満、面倒、好きから価値の種を見つける |
| 3 | アイデア作り | Idea Builder / アイデアビルダー | 誰の何を解決するかを言える |
| 4 | PMF | Experiment Designer / 実験デザイナー | 本当に欲しい人がいるか小さく試す |
| 5 | ピッチ | Pitch Challenger / ピッチチャレンジャー | 3分でアイデアを説明できる |

## Day1

- ミッション: お金と仕事の正体を見破れ。
- 成果物: 「仕事とは、____ を ____ に変えること。」
- 子供に残したい一言: 仕事は、誰かの状態を少し良くすること。

## Day2

- ミッション: 日常に隠れた価値の種を見つけろ。
- 成果物: 困りごとリスト、3人への質問。
- 子供に残したい一言: 困りごとは、価値の種。

## Day3

- ミッション: 誰の何を助けるかを形にせよ。
- 成果物: 起業の基本文。
- 子供に残したい一言: 良いアイデアは、相手の変化が見える。

## Day4

- ミッション: 本当に欲しい人がいるか小さく試せ。
- 成果物: 小さな実験計画。
- 子供に残したい一言: 完璧より、小さく試して反応を見る。

## Day5

- ミッション: 3分で価値の変化を伝えろ。
- 成果物: 3分ピッチ原稿、セミナーで見るポイント。
- 子供に残したい一言: すごさより、誰の何をどう良くするか。

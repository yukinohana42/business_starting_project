# Reward and Badge Guidelines

## 基本方針

バッジ、ポイント、スタンプは、子供を報酬で釣るために使いません。学びの行動を見える形で残すために使います。

## 必ず守ること

- 報酬で釣らない。
- 勝敗を強調しない。
- 点数で子供を序列化しない。
- バッジは能力評価ではなく、学びの行動の記録にする。
- 親が褒めるときは、結果ではなく、観察、問い、試したことを褒める。

## バッジにしてよい行動

- 身近な困りごとを見つけた。
- 誰が困っているかを具体的に言えた。
- もとの状態と変わった後を言えた。
- 相手に聞く質問を作れた。
- 小さく試す方法を考えた。
- 赤信号を次の改善に変えた。
- 自分の言葉でまとめを書いた。

## バッジにしないこと

- 一番稼げそう。
- 一番すごいアイデア。
- 親が気に入った答え。
- 早く終わった。
- ほかの子より点数が高い。

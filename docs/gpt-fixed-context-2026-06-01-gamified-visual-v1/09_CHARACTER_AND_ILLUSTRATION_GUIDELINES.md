# Character and Illustration Guidelines

## 目的

人物イラストは、抽象的な「価値」「仕事」「困りごと」「PMF」を、人物の表情、姿勢、行動の変化として見せるために使います。

## 必ず守ること

- 完全オリジナルキャラクターにする。
- 既存作品、既存キャラクター、特定漫画家の絵柄を真似しない。
- 中2男子が感情移入しやすいが、幼稚にしすぎない。
- 親が見ても安心できる服装、表情、言葉にする。
- 会社員や起業しない選択を悪く見せない。

## 推奨キャラクター

### 価値探偵ユウ

- 中学生。
- 観察メモや探偵ノートを持つ。
- 「それ、誰がどう変わる？」と考える役。

### アイデアビルダー・ソラ

- 中学生。
- ラフスケッチやカードを使って考える。
- 「まず試してみよう」と動く役。

### ナビゲーター・ミホ

- 年上すぎない親しみやすい案内役。
- 先生っぽくなりすぎない。
- 困ったときにヒントを出す役。

## Before/After

- 困った顔 -> 助かった顔。
- だるそう -> ラクそう。
- 退屈 -> ワクワク。
- わからない -> できそう。
- ひとりで困る -> 友達と盛り上がる。

Beforeは相手を笑いものにしない。Afterは大げさな成功ではなく、少し良くなった状態にする。

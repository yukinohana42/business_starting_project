# Character and Illustration Guidelines

## 目的

人物イラストは、抽象的な「価値」「仕事」「困りごと」「PMF」を、人物の表情、姿勢、行動の変化として見せるために使います。

## 必ず守ること

- 完全オリジナルキャラクターにする。
- 既存作品、既存キャラクター、特定漫画家の絵柄を真似しない。
- 中2男子が感情移入しやすいが、幼稚にしすぎない。
- 親が見ても安心できる服装、表情、言葉にする。
- 会社員や起業しない選択を悪く見せない。

## 推奨キャラクター

### 価値探偵ユウ

- 中学生。
- 好奇心が強い。
- 観察メモや探偵ノートを持つ。
- 「それ、誰がどう変わる？」と考える役。

### アイデアビルダー・ソラ

- 中学生。
- ラフスケッチやカードを使って考える。
- 「まず試してみよう」と動く役。

### ナビゲーター・ミホ

- 年上すぎない親しみやすい案内役。
- 先生っぽくなりすぎない。
- 困ったときにヒントを出す役。

## 服装・表情・ポーズ・小物

- 服装: 普通の中学生が着そうなカジュアル服。制服に寄せすぎない。
- 表情: 困る、気づく、試す、安心する、少しワクワクする。
- ポーズ: 観察する、メモする、カードを並べる、友達と話す。
- 小物: 探偵ノート、ミッションカード、付箋、ペン、スマホ、ラフスケッチ。
- 避ける小物: お金の束、高級品、派手な勝利トロフィー。

## Before/After

- 困った顔 -> 助かった顔。
- だるそう -> ラクそう。
- 退屈 -> ワクワク。
- わからない -> できそう。
- ひとりで困る -> 友達と盛り上がる。

Beforeは相手を笑いものにしない。Afterは大げさな成功ではなく、少し良くなった状態にする。

## Day1での使い方

- Page 1: 価値探偵ユウが探偵ノートを持ち、今日のミッションを見る。
- Page 2: 困っている人と助かった人のBefore/Afterを見せる。
- Page 3: お金はありがとうが見える形だと表情で見せる。
- Page 4: ユウとソラが価値変換カードを並べる。
- Page 5: ビジネス探偵カードで身近な仕事を観察する。
- Page 6: Final Missionとして自分の言葉で1文を書く。

# Day1 v2 Brushup Plan

## 目的

既存の6枚構成を保ちながら、ゲーミフィケーション、同世代オリジナルキャラクター、Before/After表現、マンガ的な1コマ表現を加えます。

## v2で変えること

- Page 1を「今日のミッション」化する。
- Page 5を「ビジネス探偵ミッション」化する。
- Page 6を「Final Mission」化する。
- 価値探偵ユウ、アイデアビルダー・ソラ、ナビゲーター・ミホを完全オリジナルキャラクターとして使う。
- 価値を「人の表情・姿勢・行動の変化」で見せる。
- 吹き出しは1ページ1つまで、短く使う。

## v1の良い点

- 仕事、お金、価値の説明順がわかりやすい。
- 既存v1は16:9と4:5の両方でレンダリング済み。
- 文字量を絞り、Zoom共有時の読みやすさを意識している。
- SOURCE_OF_TRUTHと矛盾せず、起業やお金を煽っていない。

## v1の課題

- ミッション感が弱く、子供が「今日やること」をつかむ入口を強化したい。
- 同世代キャラクターがいないため、自分たちの話として見える余地がある。
- Before/Afterの人物変化をさらに明確にしたい。
- Page 5の例が多く、画面によっては重く見える可能性がある。

## ページ別方針

1. Page 1: ユウが探偵ノートとミッションカードを持つ。「お金と仕事の正体を見破れ」。
2. Page 2: 困った顔から助かった顔へのBefore/After。
3. Page 3: ありがとうが見える形としてのお金。
4. Page 4: ユウとソラが価値変換カードを並べる。
5. Page 5: ビジネス探偵ノートで身近な仕事を観察する。
6. Page 6: Final Missionとして自分の言葉で1文を書く。

## PNG再生成前に確認すること

- 既存作品、既存キャラクター、特定漫画家の絵柄を指定していないか。
- キャラクターは完全オリジナルとして説明されているか。
- バッジやミッションが報酬や勝敗に見えないか。
- 16:9と4:5の両方で文字量が増えすぎていないか。
- Page 5は5例版と3例版のどちらがZoomで読みやすいか。
- 親が見ても安心でき、起業を押し付けていないか。

# Visual Storytelling Guidelines

## 基本方針

言葉だけで説明せず、人物の変化で価値を見せます。価値は商品やサービス名そのものではなく、受け取った人の状態が少し良くなることです。

## 抽象語の言い換え

- 価値提供 -> 困ったを助かったに変える。
- 顧客 -> それを本当に欲しい具体的な人。
- PMF -> 作ったものと欲しい人の気持ちがピタッとはまること。
- 実験 -> 完璧に作る前に、小さく試してみること。

## ページ設計

- 1ページ1メッセージ。
- 人物は基本1から2人、多くても3人。
- メインイラスト1つ、補助イラスト1から2つまで。
- 図解は関係や流れを見せる。
- カードは子供が考える入口を見せる。
- 矢印は変化を見せる。

## Zoomでの読みやすさ

- 小さい表情に頼りすぎない。
- 表情、言葉、矢印を組み合わせる。
- 吹き出しは短くする。
- 画面共有で読めない小さな文字を入れない。

# Session Gamification Map

## Day1: Value Detective / 価値探偵

- ミッション: お金と仕事の正体を見破れ。
- ゴール: 仕事・お金・価値を相手の変化で見る。
- 使うカード: 価値変換カード、ビジネス探偵カード、変化カード。
- Final Mission: 仕事とは、＿＿を＿＿に変えること。
- 親の問い: 誰がどう変わった？
- 禁止: 稼げる仕事当て、金額ランキング。

## Day2: Problem Hunter / 困りごとハンター

- ミッション: 日常に隠れた価値の種を見つけろ。
- ゴール: 日常の不満、面倒、好き、欲しい気持ちを観察する。
- 使うカード: 困りごと発見カード、観察カード、発見カード。
- Final Mission: 価値の種は、＿＿の中にある。
- 親の問い: いつ困る？今はどうしている？
- 禁止: 困っている人を笑う、個人情報を聞き出す。

## Day3: Idea Builder / アイデアビルダー

- ミッション: 誰の何を助けるかを形にせよ。
- ゴール: 顧客、問題、解決案を1つの文で言える。
- 使うカード: 顧客カード、解決案カード、価値変換カード。
- Final Mission: 私たちは、＿＿の＿＿を＿＿に変える。
- 親の問い: それは誰のため？使った後、その人はどう変わる？
- 禁止: すごい案だけを褒める。

## Day4: Experiment Designer / 実験デザイナー

- ミッション: 本当に欲しい人がいるか小さく試せ。
- ゴール: PMFを理解し、小さな実験を考える。
- 使うカード: PMF信号カード、実験カード、反応メモカード。
- Final Mission: 最初に試す一番小さいことは、＿＿。
- 親の問い: 何を見たら欲しいとわかる？赤信号ならどこを変える？
- 禁止: 赤信号を失敗扱いする、無理に販売させる。

## Day5: Pitch Challenger / ピッチチャレンジャー

- ミッション: 3分で価値の変化を伝えろ。
- ゴール: 問題、顧客、解決案、試し方を3分で伝える。
- 使うカード: ピッチ問いカード、価値変化カード、聞き手メモカード。
- Final Mission: 私たちのアイデアは、＿＿を＿＿に変える。
- 親の問い: 一番伝えたい変化は何？
- 禁止: 発表の勝ち負け、親の採点会。

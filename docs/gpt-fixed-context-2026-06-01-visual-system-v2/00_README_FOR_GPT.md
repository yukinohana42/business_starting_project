# README for GPT

This folder is the current fixed context for the junior entrepreneurship workshop project.

Use this folder at the start of a new GPT/Codex thread when working on curriculum, slides, cards, worksheets, parent materials, prompts, or production planning.

## Most Important Rule

Preserve the Source of Truth first.

The project does not push entrepreneurship as an obligation. It helps junior high school students learn to observe value, talk with people, try small experiments, and improve. Employment, non-entrepreneurship, and ordinary work are never shown as inferior.

## Current Visual Direction

The child-facing materials are not ordinary explanation slides.

They must be designed as discovery-game learning experiences where the student can see what to do:

- observe
- choose
- compare
- write
- ask
- test
- reflect

Game elements are learning scaffolds, not rewards or competition.

## Critical Lesson From Day1 v2

Day1 16:9 v2 PNGs were readable but did not fully achieve the intended visual direction. The main issue was that the visual system allowed cards, round character icons, and explanatory layout to pass as acceptable.

For v3 and later, do not proceed to HTML/CSS/PNG until each page has:

- a clear screen type such as Mission Board, Case File, Evidence Board, Card Table, Experiment Lab, Pitch Stage, or Final Mission Sheet
- visible character action, not only a face or name label
- a game element that supports a real learning action
- a visible Before/After or learning change when the concept requires it
- a review against the global visual acceptance criteria

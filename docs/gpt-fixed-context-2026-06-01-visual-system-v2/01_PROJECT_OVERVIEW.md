# Project Overview

## What This Project Creates

This repository creates a home-based entrepreneurship workshop curriculum for junior high school students.

The materials include:

- implementation plans
- student-facing infographics or slides
- printable cards
- worksheets
- parent facilitator materials
- prompts and production instructions for future Codex/GPT work

## Target Learner

The main learner is a second-year junior high school student.

The workshop assumes no prior business knowledge. It should feel safe, concrete, warm, and practical.

## Parent Role

The parent is not a teacher, scorer, judge, or game master.

The parent acts as a facilitator who:

- asks good questions
- listens without rushing to correct
- helps the student observe real people
- keeps the activity small and safe
- praises the process of noticing, asking, trying, and improving

## Learning Goal

The student learns that:

- work means providing value to someone
- money is a visible sign that value was received
- value is seen in another person's change
- ideas come from observation and conversation
- small experiments are safer and more useful than big guesses
- a pitch is a way to explain value clearly, not a contest

## Visual Experience Goal

The student-facing experience should not look like a normal explanatory slideshow.

It should look and function like a discovery-game learning experience:

- Mission Board
- Quest Map
- Detective Note
- Case File
- Evidence Board
- Card Table
- Experiment Lab
- Pitch Stage
- Final Mission Sheet
- Reflection Log

The student should be able to tell from the screen what action to take next.

## Safety and Ethics

The materials must not:

- pressure the student to become an entrepreneur
- present employment or non-entrepreneurship as inferior
- glamorize money or profit
- use ranking, scores, winning, failure, or reward bait
- imitate existing IP, existing characters, or a specific artist's style
- turn badges, stamps, or cards into external rewards

## Current State

Day1 v2 planning, 16:9 HTML/CSS, PNGs, and contact sheet exist, but the v2 visual result has not been accepted as the final direction.

The current work adds a stronger global visual system, acceptance criteria, prompt updates, and v3 design briefs before any new production.

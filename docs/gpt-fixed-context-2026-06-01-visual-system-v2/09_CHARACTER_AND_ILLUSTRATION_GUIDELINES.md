# キャラクター・人物イラスト方針

## 目的

人物イラストは、教材を飾るためだけに使いません。抽象的な「価値」「仕事」「困りごと」「PMF」を、人物の表情、姿勢、行動の変化として見せるために使います。

中学2年生に近いキャラクターを使うことで、子供が「自分たちの話かも」と感じやすくします。ただし、幼稚にしすぎず、親が見ても安心できる落ち着いた表現にします。

## 必ず守ること

- 完全オリジナルキャラクターにする。
- 既存作品、既存キャラクター、特定漫画家の絵柄を真似しない。
- ユーザーが例に挙げた既存作品名は、既存IPを模倣する意味ではない。
- キャラクターは中2男子が感情移入しやすいが、子供っぽくしすぎない。
- 親が見ても安心できる服装、表情、言葉にする。
- 会社員や起業しない選択を悪く見せない。
- 不安を煽る表現を入れない。

## 使うべき役割

### 価値探偵ユウ

- 中学生。
- 好奇心が強い。
- 観察メモや探偵ノートを持つ。
- 「それ、誰がどう変わる？」と考える役。
- Day1とDay2で使いやすい。

### アイデアビルダー・ソラ

- 中学生。
- ラフスケッチやカードを使って考える。
- 「まず試してみよう」と動く役。
- Day3とDay4で使いやすい。

### ナビゲーター・ミホ

- 年上すぎない親しみやすい案内役。
- 先生っぽくなりすぎない。
- 困ったときにヒントを出す役。
- 親の代わりに正解を言うのではなく、問いを出す。

名前や設定は改善してよいですが、完全にオリジナルにします。

## 使ってはいけない表現

- 既存漫画・アニメのキャラクターに似せる。
- 特定作品の衣装、髪型、決めポーズ、ロゴを連想させる。
- バトル、勝敗、ランキングを強く見せる。
- お金を持って喜ぶだけの表現。
- 「会社員は負け」のような表現。
- 子供を焦らせる表情や怖い警告。
- 過度に幼児向けのデフォルメ。

## 服装・表情・ポーズ・小物

- 服装: 普通の中学生が着そうなカジュアル服。制服に寄せすぎない。
- 表情: 困る、気づく、試す、安心する、少しワクワクする。
- ポーズ: 観察する、メモする、カードを並べる、友達と話す。
- 小物: 探偵ノート、ミッションカード、付箋、ペン、スマホ、ラフスケッチ。
- 避ける小物: お金の束、高級品、派手な勝利トロフィー。

## 多様性への配慮

- 性別役割を固定しない。
- 得意な子だけが活躍する見せ方にしない。
- 話すのが得意、書くのが得意、観察が得意など、複数の入り口を見せる。
- 体格、髪型、服装は自然な幅を持たせる。

## 表情の使い方

価値は、人物の変化として見せます。

- 困った顔 -> 助かった顔。
- だるそう -> ラクそう。
- 退屈 -> ワクワク。
- わからない -> できそう。
- ひとりで困る -> 友達と盛り上がる。

## Before/Afterの見せ方

- Beforeは相手を笑いものにしない。
- Afterは大げさに勝利させず、少し良くなった状態にする。
- 矢印やカードで「何が変わったか」を添える。
- 変化は、表情、姿勢、行動、周囲との関係で見せる。

## Day1での使い方

- Page 1: 価値探偵ユウが探偵ノートを持ち、今日のミッションを見る。
- Page 2: 困っている人と助かった人のBefore/Afterを見せる。
- Page 3: お金はありがとうが見える形だと、受け取る人の表情で見せる。
- Page 4: 価値変換カードをユウが並べる。
- Page 5: 身近な仕事をビジネス探偵カードで観察する。
- Page 6: Final Missionとして、ユウとソラが自分の言葉で1文を書く。

## v2追加: 丸アイコンだけでは不可

キャラクターは、顔アイコンや名前ラベルではありません。思考の代理人として行動している必要があります。

合格する行動:

- メモする。
- 観察する。
- 指差す。
- カードを並べる。
- 付箋を書く。
- ラフスケッチする。
- 質問する。
- 試す。
- 反応を見る。
- 振り返る。
- 発表する。

不合格:

- 丸アイコンだけ。
- 顔だけ。
- 名前ラベルだけ。
- 立っているだけ。
- 同じ顔を色違いで使うだけ。

HTML/CSSで作る場合も、手元、姿勢、小物、視線、行動が見える表現を優先します。必要ならinline SVG、手描き風SVG、または完全オリジナルの画像生成素材を使います。

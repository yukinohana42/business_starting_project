# 親向け説明

## このプロジェクトの目的

このプロジェクトは、子供に起業を押し付けるものではありません。

これからの時代に、会社に入るとしても、自分で何かを始めるとしても役立つ「価値を作る力」を育てるための家庭内ワークショップです。

## 子供に身につけてほしいこと

- 誰かにとって価値あることを見つける
- 困っている人の話を聞く
- 自分の思いつきを相手目線で見直す
- 小さく試して反応を見る
- うまくいかなかった時に改善する

## 親の役割

親は、正解を教える先生ではありません。問いを出し、子供の考えを広げる人です。

良い関わり方:

- 「それは誰が困っている？」
- 「本当に使いたい人はいるかな？」
- 「今はどうやって解決している？」
- 「まず小さく試すなら何ができる？」
- 「反応が悪かったら、何を変える？」

避けたい関わり方:

- 「それは無理」
- 「もっと稼げるものにしなさい」
- 「正解はこれ」
- 「起業しないと将来困る」

## 実施後に見るポイント

子供が完璧なアイデアを出せたかより、次を見ます。

- 自分で問いを出せたか
- 具体的な誰かを思い浮かべたか
- 相手に聞いてみようとしたか
- 小さく試す案を出せたか
- 反応から学べたか

# マンガ的1コマ表現の使い方

## 基本方針

マンガ的表現は、理解補助として使います。既存漫画作品の表現やキャラクターを模倣するためではありません。

目的は、抽象的な価値や変化を、短い場面として見せることです。

## 使ってよい場面

- 困ったが助かったに変わる瞬間。
- 面倒がラクになる瞬間。
- わからないことが、できそうに変わる瞬間。
- 友達と話して発見が生まれる瞬間。
- Final Missionで自分の言葉にする瞬間。

## 吹き出し

- 吹き出しは短くする。
- 1つの吹き出しは1文までを基本にする。
- 中学生がそのまま言いそうな言葉にする。
- 親向け説明を吹き出しに詰め込まない。

例:

- 「それ、誰が助かる？」
- 「面倒がラクになるってことか」
- 「まず小さく試してみよう」
- 「仕事って、変化を作ること？」

## 擬音

- 擬音は多用しない。
- 使う場合は小さく、理解を助ける範囲にする。
- 派手なバトル風、勝利演出風にはしない。

## コマ数

- 1ページに1から2コマまで。
- コマを増やしすぎず、図解と組み合わせる。
- 4:5版では1コマを大きめに見せる。
- 16:9版ではコマとカードを横並びにしやすい構成にする。

## 教材独自の視覚言語

既存作品を真似るのではなく、この教材独自の道具を使います。

- 探偵ノート
- ミッションカード
- 価値変換カード
- ビジネス探偵カード
- 発見メモ
- Final Mission欄
- 進捗マップ

## 避けること

- 既存漫画作品の表現を再現する。
- 既存キャラクターに似せる。
- 特定漫画家の絵柄を指定する。
- バトル、勝利、敗北を強く見せる。
- 子供っぽくなりすぎるデフォルメ。
- 親が見て不安になる煽り表現。

## 親が見ても安心できる表現

- 温かく、落ち着いた表情にする。
- 失敗を責めない。
- 「まずやってみる」「聞いてみる」「変えてみる」を前向きに見せる。
- お金はありがとうの形として扱う。
- 起業は選択肢の1つであり、押し付けない。

## Day1での使い方

- Page 1: 価値探偵ユウがミッションカードを受け取る1コマ。
- Page 2: 困った顔から助かった顔へのBefore/After。
- Page 3: 「ありがとう」が届く小さな1コマ。
- Page 4: 価値変換カードを並べる1コマ。
- Page 5: 身近な仕事を観察する探偵ノート。
- Page 6: Final Missionに自分の言葉を書き込む1コマ。

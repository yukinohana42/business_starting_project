# README for GPT

## このフォルダの目的

このフォルダは、「中学2年生向け・家庭内起業ワークショップ準備プロジェクト」を次回以降のChatGPTスレッドに引き継ぐための固定情報源です。

このフォルダだけ読めば、プロジェクトの思想、設計、教材の配置、現状、次にやるべきことがわかる状態を目指しています。

## 読む順番

2. `02_SOURCE_OF_TRUTH.md`
3. `01_PROJECT_OVERVIEW.md`
4. `07_CURRICULUM_5_SESSIONS.md`

## このプロジェクトの一言説明

中学2年生の子供たちが起業セミナーに参加する前に、家庭で5回の準備ワークショップを行い、「誰かにとって価値あることを見つけ、小さく試す力」を育てるプロジェクトです。

## 対象者

- 中学2年生の男の子2人
- 起業やビジネスの前提知識はほぼない
- 座学だけだと飽きやすい
- ゲーム、部活、スマホ、勉強、友達、親、文化祭、お店など身近な題材が入りやすい
- 進行役は主に親

## 重要な教育方針

まずは `02_SOURCE_OF_TRUTH.md` を守ってください。

このプロジェクトは、子供に起業を押し付けるものではありません。会社に入るとしても、自分で何かを始めるとしても役立つ「価値を作る力」を育てるためのものです。

## 今後ChatGPTに依頼できる作業例

- 第1回の台本を父親がそのまま読める形にする
- 子供向けスライドを実施可能レベルに磨く
- カード教材をA4印刷用に整える
- ワークシートを記入しやすいレイアウトにする
- 親向け説明を10分で話せる構成にする
- 実施後メモから次回改善案を作る

## 最重要ルール

まずは `SOURCE_OF_TRUTH` を守ること。

起業を押し付けない。不安を煽らない。お金を下品に扱わない。価値は相手の変化として扱う。親は答えを教える人ではなく、問いを出すファシリテーターとして扱う。

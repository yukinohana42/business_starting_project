# Skills Overview

## workshop-designer

用途:

- 新しいワークショップ回を設計する
- 既存回を30分版、45分版、60分版に調整する

主な出力:

- 目的、対象、伝える概念、導入、メインワーク、問い、まとめ、宿題、必要なカードやワークシート

## slide-writer

用途:

- 子供向けまたは親向けのMarkdownスライド原稿を作る

ルール:

- 1スライド1メッセージ
- 短い問いを入れる
- 身近な例を使う
- 専門用語は言い換える
- スライド区切りは `---`

## card-designer

用途:

- 印刷用カード教材を作る

カード種類:

- 価値カード
- 問題カード
- 顧客カード
- アイデアカード
- PMF信号カード
- ピッチ問いカード

## parent-briefing-writer

用途:

- 母親、父親、家族向け説明資料を作る

書くこと:

- プロジェクトの目的
- 起業を押し付けるものではないこと
- 育てたい力
- 親の役割
- 子供への良い問い
- 避けたい言い方

## facilitator-script-writer

用途:

- 親が当日読める進行台本を作る

ルール:

- 親が話しすぎない
- 5から10分ごとに子供が話す、書く、選ぶ時間を入れる
- 否定せずに問いで深める
- 最後にまとめと次回へのつなぎを入れる

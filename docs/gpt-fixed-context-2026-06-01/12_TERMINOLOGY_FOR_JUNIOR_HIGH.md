# Terminology for Junior High

## 中学生向け言い換え辞書

| 用語 | 言い換え |
|---|---|
| 仕事 | 誰かの状態を少し良くすること |
| お金 | 価値を受け取った人からのありがとうが見える形になったもの |
| 価値 | 相手の変化 |
| 価値提供 | 誰かの困ったを助かったに変えること |
| 顧客 | それを本当に欲しい人 |
| アイデア | 誰かの困りごとや欲しい気持ちを解決する方法 |
| PMF | 作ったものと欲しい人の気持ちがピタッとはまること |
| 市場調査 | 人の本音を探ること |
| 検証 | 小さく試して反応を見ること |
| 競合 | その人が今すでに使っている別の方法 |
| 実験 | 完璧に作る前に、小さく試してみること |

## 使いやすい例

- ラーメン屋: お腹が空いたを満足に変えている。
- コンビニ: 今すぐ欲しいを、すぐ買えたに変えている。
- YouTuber: 退屈を楽しいに変えている。
- 勉強アプリ: わからないを、できそうに変えている。

# Project Overview

## プロジェクトの目的

このプロジェクトは、中学2年生の子供たちが起業セミナーに参加する前に、家庭で4から5回の事前ワークショップを実施するための教材一式を作るものです。

目的は、起業家になることを押し付けることではありません。誰かにとって価値あることを見つけ、形にし、小さく試せる力を育てることです。

## 対象者

- 中学2年生の男の子2人
- 起業やビジネスの専門知識は前提にしない
- 身近な題材から入る
- 親が家庭で進行する

## 実施形式

- 回数: 標準5回、圧縮版4回
- 時間: 各30から60分
- 場所: 家庭のリビングやダイニング
- 道具: スライド、カード、ワークシート、紙、ペン

## 最終ゴール

子供たちが起業セミナーに参加するときに、ただ話を聞くだけではなく、次の視点を持つことです。

- 仕事は誰かへの価値提供である
- 価値は相手の変化である
- 顧客は具体的な誰かである
- 良いアイデアは相手が本当に欲しいものである
- 小さく試して反応を見る

## 作成する教材

- 親向け説明資料
- 子供向けスライド
- 各回の進行台本
- カード教材
- ワークシート
- 振り返りシート
- Zoom共有用インフォグラフィック
- Day別の実施パッケージ
- ChatGPT/Codex作業用プロンプト

## 5回構成と4回構成の関係

標準は5回構成です。

1. 働く・お金・価値
2. 価値の見つけ方
3. アイデア作り
4. PMFと小さな実験
5. 3分ピッチ

4回に圧縮する場合は、第3回「アイデア作り」と第4回「PMF」を統合します。

## 現在の強化対象

成果物は `outputs/day1/` にあります。

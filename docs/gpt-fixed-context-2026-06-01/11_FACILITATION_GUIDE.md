# Facilitation Guide

## 基本姿勢

親は話しすぎません。子供が考える時間、書く時間、選ぶ時間、話す時間を作ります。

目安は、親が話す時間3割、子供が動く時間7割です。

## 否定しない

子供の案が粗くても、すぐに否定しません。

避ける言い方:

- それは無理
- そんなの誰も使わない
- もっとちゃんと考えて

使う言い方:

- 誰が使いそう？
- 今はどうやって解決している？
- 小さく試すなら何ができる？

## 5から10分ごとに子供が動く設計にする

親の説明が長く続くと飽きやすいので、5から10分ごとに子供が何かをします。

- 書く
- 話す
- 選ぶ
- カードを引く
- 点数をつける
- 発表する

## 使うべき質問例

- それは誰が困っている？
- どれくらい困っている？
- いつ困っている？
- 今はどうしている？
- その方法の何が面倒？
- 使うとどんな気持ちになる？
- 明日できる一番小さい試し方は？

## 子供の反応が薄いとき

- 自分の話ではなく、友達や親の例に変える
- カードを使って選ばせる
- 書く前に話すだけにする
- 例を1つ出してから、似たものを探させる

## 盛り上がったときの深掘り

- それは誰の何を良くしている？
- 2回目も使いたいと思う？
- 友達にすすめたい？
- その人は今何を使っている？
- まず1回だけ試すならどうする？

# Source of Truth

このファイルは、このプロジェクトで絶対にぶらさない思想です。

## 基本思想

- 仕事とは、誰かへの価値提供である。
- お金は、価値提供に対する対価である。
- 価値は、相手の変化である。
- ビジネスの種は、不満、面倒、好き、快感、欲求の中にある。
- 顧客は「みんな」ではなく、具体的な誰かである。
- 良いアイデアとは、自分が作りたいものではなく、相手が本当に欲しいものである。
- PMFは、作ったものと欲しい人の気持ちがピタッとはまることである。
- 起業は、完璧に考えるより、小さく試して、反応を見て、改善するものである。

## 親の役割

親は教える人というより、問いを出すファシリテーターです。

親が先に正解を言うと、子供は考えるより「親が喜ぶ答え」を探しやすくなります。親は、答えを教えるより、子供の観察と発見を助けます。

使いたい問い:

- それは本当に使いたい人がいるかな？
- 誰が一番困っていると思う？
- 今はどうやって解決している？
- 小さく試すなら、何からできる？
- 反応が悪かったら、どこを変える？

## お金の扱い方

お金の話は、下品なものではなく、価値が届いたサインとして扱います。

お金は「価値を受け取った人からのありがとうが見える形になったもの」と説明します。

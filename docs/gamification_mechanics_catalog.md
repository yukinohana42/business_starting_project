# ゲーミフィケーション仕組みカタログ

このカタログは、教材に入れてよい仕組みと、避けるべき使い方を整理するためのものです。

## 基本ルール

- 仕組みは学びを助けるために使う。
- 勝敗、順位、報酬で子供を動かさない。
- 「発見した」「聞けた」「試した」「言葉にできた」を記録する。
- うまくいかなかったことも、次に変えるサインとして扱う。

## 仕組み一覧

### ミッション

- 目的: 今日やることを短く見せる。
- 使いどころ: 各回の導入、ワーク開始前、まとめ前。
- 子供向けの見せ方: 「今日のミッション: お金と仕事の正体を見破れ」のように短く出す。
- 注意点: 大げさにしすぎず、学習目的とつなげる。
- 避けるべき使い方: クリアできないと失敗、という見せ方。

### クエスト

- 目的: 小さな行動を選びやすくする。
- 使いどころ: カードを選ぶ、身近な例を探す、インタビュー質問を作る場面。
- 子供向けの見せ方: 「3つの困りごとを見つけるクエスト」。
- 注意点: 作業量を増やしすぎない。
- 避けるべき使い方: 宿題を大量に増やす口実にする。

### レベル

- 目的: 5回全体の見通しを作る。
- 使いどころ: 進捗マップ、各回の役割説明。
- 子供向けの見せ方: Day1は価値探偵、Day2は困りごとハンターのように役割を変える。
- 注意点: 能力の上下ではなく、扱うテーマの段階として見せる。
- 避けるべき使い方: 子供ごとの優劣を示す。

### バッジ

- 目的: 学びの行動を記録する。
- 使いどころ: セッションの最後、振り返り、親の声かけ。
- 子供向けの見せ方: 「観察メモを書けた」「相手の変化を言えた」など。
- 注意点: 能力評価ではなく、行動の記録にする。
- 避けるべき使い方: バッジ数で順位をつける。

### ステッカー

- 目的: ワークシートやカードに小さな達成感を出す。
- 使いどころ: 発見メモ、問いカード、Final Mission欄。
- 子供向けの見せ方: 「発見」「問い」「試した」の小さな印。
- 注意点: かわいくしすぎず、中学生が照れない表現にする。
- 避けるべき使い方: ご褒美シールだけを目的にする。

### 進捗マップ

- 目的: 5回全体の現在地を見せる。
- 使いどころ: 各回の冒頭と最後。
- 子供向けの見せ方: 価値探偵からピッチチャレンジャーまでの道筋。
- 注意点: 競争の順位表にしない。
- 避けるべき使い方: 遅れている人を目立たせる。

### 発見カード

- 目的: 生活の中の気づきを拾いやすくする。
- 使いどころ: Day1、Day2の導入とワーク。
- 子供向けの見せ方: 「困った」「面倒」「好き」「欲しい」を選ぶカード。
- 注意点: 正解カードにしない。
- 避けるべき使い方: 親が正しいカードを選ばせる。

### ビジネス探偵カード

- 目的: 身近な仕事を「相手の変化」で見る。
- 使いどころ: Day1の身近な例、Day2の観察。
- 子供向けの見せ方: 「誰が」「何に困って」「どう変わった」をメモするカード。
- 注意点: 金額や儲けだけに寄せない。
- 避けるべき使い方: 稼げる仕事ランキングにする。

### 価値変換カード

- 目的: もとの状態と変わった後をつなげる。
- 使いどころ: Day1の価値変換ゲーム、Day3のアイデアづくり。
- 子供向けの見せ方: 「面倒 -> ラク」「わからない -> できそう」。
- 注意点: 変化を具体的な人に戻す。
- 避けるべき使い方: かっこいい言葉だけを作る。

### Final Mission

- 目的: その回の学びを自分の言葉にする。
- 使いどころ: 各回の最後。
- 子供向けの見せ方: Day1なら「仕事とは、＿＿を＿＿に変えること」。
- 注意点: 模範解答を押し付けない。
- 避けるべき使い方: テストの正解欄にする。

### XP

- 目的: 使う場合は経験の記録として扱う。
- 使いどころ: 長期の進捗表や振り返り。
- 子供向けの見せ方: 「質問したXP」「観察したXP」のように行動名を添える。
- 注意点: 点数競争になりやすいので慎重に使う。
- 避けるべき使い方: 合計点で順位を出す。

### ふりかえりスタンプ

- 目的: 今日の学びを短く残す。
- 使いどころ: セッション終了時、ワークシート最下部。
- 子供向けの見せ方: 「発見」「気になる」「次に試す」の3種類程度。
- 注意点: 感想を無理にポジティブにしない。
- 避けるべき使い方: よい感想だけを選ばせる。

## 使うときの判断基準

- 子供が次の一言を言いやすくなるか。
- 親が問いを出しやすくなるか。
- 価値を相手の変化として見せられるか。
- 競争、報酬、煽りに寄っていないか。

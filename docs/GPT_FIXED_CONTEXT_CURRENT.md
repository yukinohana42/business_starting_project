## Current Meaning of This Fixed Context

This fixed context updates the project from a broad gamified-visual foundation to a stricter Visual Learning System v2.

The most important correction is that child-facing materials must not become ordinary explanation slides with cards, round character icons, and game-flavored labels. They must work as discovery-game learning screens where the student can see what to do next.

## Project Rule

Preserve the Source of Truth first. Do not push entrepreneurship as an obligation. Help the students learn how to find value, talk to people, test small, and improve. Gamification is a scaffold for learning, not the goal. Use only fully original characters and never imitate existing IP.

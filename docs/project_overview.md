# プロジェクト概要

## 何を作るか

家庭で実施できる、起業セミナー前の準備ワークショップ一式を作ります。

作るものは、進行台本、子供向けスライド、カード教材、ワークシート、親向け説明資料です。

## 誰のためか

主な対象は中学2年生の男の子2人です。起業やビジネスの専門知識は前提にしません。

補助対象は、家庭で進行する親です。親は一方的な先生ではなく、子供の観察と発見を助ける役割です。

## 何を学ぶか

- 仕事は誰かへの価値提供である
- お金は価値提供への対価である
- 価値は相手の変化である
- 困りごと、面倒、好き、欲しい気持ちの中に価値の種がある
- 顧客は「みんな」ではなく具体的な誰かである
- アイデアは思いつきだけでなく、観察とインタビューから作る
- 小さく試し、反応を見て、改善する

## 実施イメージ

- 回数: 4から5回
- 時間: 各30から60分
- 場所: 家庭のリビング、ダイニング、オンラインでも可
- 道具: 紙、ペン、カード、スライド、スマホでの簡単な調査

## 成果物

最終的に、子供たちは起業セミナーに参加する前に、次のものを持った状態になります。

- 自分たちのミニ起業アイデア
- 誰のどんな困りごとを解決するかの説明
- 小さく試す方法
- 3分ピッチの原稿
- セミナーで見るべきポイント

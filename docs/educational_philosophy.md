# 教育方針

## 起業教育としての位置づけ

この教材は、会社を作る手続きやお金儲けの技術を教えるものではありません。

中心は、価値を見つけ、作り、届ける力です。

## 育てたい力

- 観察する力
- 相手の話を聞く力
- 具体的な誰かを思い浮かべる力
- 小さく試す力
- 反応を見て変える力
- 自分の言葉で説明する力

## 避けたい授業像

- 親が長く説明する
- 子供が正解を当てる
- すごいアイデアを出すことだけを目指す
- 稼げるかどうかだけで判断する
- 将来不安を動機にする

## 目指す授業像

子供が「これ、ちょっと面白いかも」「誰かに聞いてみたい」「小さく試せるかも」と感じることを目指します。

起業という言葉を大きく見せすぎず、日常の困りごとを見つける遊びとして入ります。

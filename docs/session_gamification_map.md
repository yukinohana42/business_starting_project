# 全5回ミッションマップ

5回全体を、価値を見つけて小さく試すミッション構造として整理します。各回のミッションは、子供が入りやすくするための入口であり、学習目的そのものを置き換えるものではありません。

## 全体の流れ

| Day | ミッション名 | ゴール | 子供に残したい一言 |
|---|---|---|---|
| Day1 | Value Detective / 価値探偵 | 仕事・お金・価値を相手の変化で見る | 仕事は、誰かの状態を少し良くすること |
| Day2 | Problem Hunter / 困りごとハンター | 日常の不満・面倒・好きから価値の種を見つける | 困りごとは、価値の種 |
| Day3 | Idea Builder / アイデアビルダー | 誰の何を解決するかを形にする | 良いアイデアは、相手の変化が見える |
| Day4 | Experiment Designer / 実験デザイナー | 本当に欲しい人がいるか小さく試す | 完璧より、小さく試して反応を見る |
| Day5 | Pitch Challenger / ピッチチャレンジャー | 3分で伝える | すごさより、誰の何をどう良くするか |

## Day1: Value Detective / 価値探偵

- ミッション名: お金と仕事の正体を見破れ
- その回のゴール: 仕事は価値提供であり、お金は価値が届いたサインだと理解する。
- 子供に残したい一言: 仕事は、誰かの状態を少し良くすること。
- 使うカード: 価値変換カード、ビジネス探偵カード、変化カード。
- 使うワーク: 「仕事とは、＿＿を＿＿に変えること」の1文。
- 使えるバッジ: 変化発見バッジ、価値探偵メモバッジ。
- 親の問い: 「誰がどう変わった？」「何を売っているかではなく、何が良くなった？」
- やってはいけない演出: 稼げる仕事当て、職業の優劣づけ、金額ランキング。
- Final Mission: 仕事とは、＿＿を＿＿に変えること。

## Day2: Problem Hunter / 困りごとハンター

- ミッション名: 日常に隠れた価値の種を見つけろ
- その回のゴール: 不満、面倒、好き、欲しい気持ちを観察し、価値の種として扱える。
- 子供に残したい一言: 困りごとは、価値の種。
- 使うカード: 困りごと発見カード、観察カード、発見カード。
- 使うワーク: 困りごと10個、インタビュー質問3つ。
- 使えるバッジ: 観察メモバッジ、問いづくりバッジ。
- 親の問い: 「いつ困る？」「今はどうしている？」「それが変わるとうれしい人は誰？」
- やってはいけない演出: 困っている人を笑う、個人情報を聞き出す、SNSで知らない人に聞く。
- Final Mission: 価値の種は、＿＿の中にある。

## Day3: Idea Builder / アイデアビルダー

- ミッション名: 誰の何を助けるかを形にせよ
- その回のゴール: 顧客、問題、解決案を1つの文で言える。
- 子供に残したい一言: 良いアイデアは、相手の変化が見える。
- 使うカード: 顧客カード、解決案カード、価値変換カード。
- 使うワーク: 起業の基本文、20個アイデア出し。
- 使えるバッジ: アイデア量産バッジ、相手具体化バッジ。
- 親の問い: 「それは誰のため？」「使った後、その人はどう変わる？」
- やってはいけない演出: 大きすぎる事業計画を求める、すごい案だけを褒める。
- Final Mission: 私たちは、＿＿の＿＿を＿＿に変える。

## Day4: Experiment Designer / 実験デザイナー

- ミッション名: 本当に欲しい人がいるか小さく試せ
- その回のゴール: PMFを「作ったものと欲しい人の気持ちがピタッとはまること」と理解し、小さな実験を考える。
- 子供に残したい一言: 完璧より、小さく試して反応を見る。
- 使うカード: PMF信号カード、実験カード、反応メモカード。
- 使うワーク: 小さな実験計画、青信号・黄信号・赤信号の判断。
- 使えるバッジ: 小さく試すバッジ、反応を見るバッジ。
- 親の問い: 「何を見たら欲しいとわかる？」「赤信号ならどこを変える？」
- やってはいけない演出: 赤信号を失敗扱いする、無理に販売させる、知らない人に試させる。
- Final Mission: 最初に試す一番小さいことは、＿＿。

## Day5: Pitch Challenger / ピッチチャレンジャー

- ミッション名: 3分で価値の変化を伝えろ
- その回のゴール: 問題、顧客、解決案、試し方を3分で伝える。
- 子供に残したい一言: すごさより、誰の何をどう良くするか。
- 使うカード: ピッチ問いカード、価値変化カード、聞き手メモカード。
- 使うワーク: 3分ピッチ原稿、セミナーで見るポイント。
- 使えるバッジ: 伝えるチャレンジバッジ、改善メモバッジ。
- 親の問い: 「一番伝えたい変化は何？」「聞いた人が質問しやすい言い方は？」
- やってはいけない演出: 発表の勝ち負け、親の採点会、起業家らしさの押し付け。
- Final Mission: 私たちのアイデアは、＿＿を＿＿に変える。

## 親の立ち位置

親はゲームマスターではなく、問いを出す伴走者です。進行はミッション風にしても、評価者になりすぎないことを優先します。

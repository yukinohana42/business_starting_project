# 価値カード

印刷して切り、ランダムに引いて使うカードです。

## カード一覧

| カード | 変化の意味 | 例 |
| --- | --- | --- |
| ラクになる | 面倒が減る | 宿題の手順が見える |
| 楽しくなる | 退屈が減る | 勉強がゲームっぽくなる |
| 早くできる | 時間が短くなる | 朝の準備が早く終わる |
| 安心できる | 不安が減る | 忘れ物を防げる |
| かっこよくなる | 自分に自信が出る | 髪型や発表が決まる |
| うまくなる | 成長を感じる | 英単語を覚えやすくなる |
| 忘れにくくなる | ミスが減る | 持ち物チェックができる |
| 友達と盛り上がる | 一人より楽しい | 勉強を競争にする |
| 親に怒られにくくなる | 家庭の摩擦が減る | スマホ時間を見える化する |
| 自信が出る | やればできる感覚 | 小さな達成が見える |

## 使い方

1. 問題カードを1枚引く。
2. 価値カードを1枚引く。
3. 「この問題を、この価値に変えるには？」と考える。

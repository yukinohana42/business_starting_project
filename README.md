# 中学2年生向け・家庭内起業ワークショップ準備プロジェクト

このフォルダは、中学2年生の子供たちが起業セミナーに参加する前に、家庭で4から5回の事前ワークショップを行うための教材一式です。

目的は、子供たちに「起業家になれ」と押し付けることではありません。会社に入るとしても、自分で何かを始めるとしても、誰かにとって価値あるものを見つけ、形にし、小さく試せる人になるための練習です。

## 5回構成と4回構成

標準は5回構成です。

1. 働くって何？お金って何？価値って何？
2. 価値の見つけ方: 不満、面倒、好きの観察
3. アイデアを作る: 誰の何を解決するのか
4. PMF: 作りたいものではなく欲しいものを作る
5. ミニ起業体験の3分ピッチ

4回に圧縮する場合は、第3回と第4回を統合します。詳しくは [docs/workshop_curriculum_4_sessions.md](docs/workshop_curriculum_4_sessions.md) を見てください。

## 家庭での使い方

1回30から60分で使います。各回は「親が説明する時間」よりも「子供が考える、話す、書く、選ぶ時間」を多くします。

親の役割は先生ではなく、問いを出すファシリテーターです。正解を当てさせるより、本人の観察と発見を増やします。

## 大事な言い換え

- 仕事: 誰かの状態を少し良くすること
- お金: 価値を受け取った人からのありがとうが見える形になったもの
- 価値: 相手の変化
- 顧客: 自分たちが助けたい具体的な誰か
- 起業: 誰かの困りごとや欲しい気持ちを見つけ、解決する仕組みを作ること
- PMF: 作ったものと欲しい人の気持ちがピタッとはまること

## 注意

この教材は、子供に将来の不安を背負わせるためのものではありません。「自分で考え、誰かに聞き、小さく試し、反応から学ぶ」経験を作るためのものです。
